## Aufgabe

**Ziel:** Punktzahl in Note umwandeln und Pass/Fail bestimmen.

**Anforderungen:**
- Prompt: `Enter score (0-100):`
- Validierung: < 0 oder > 100 ‚Üí `Invalid score.`
- Notenskala:
  - 90-100: A
  - 80-89: B
  - 70-79: C
  - 60-69: D
  - 0-59: F
- D und F = Fail, sonst Pass
- Ausgabe: `Grade: <Letter> (<Pass/Fail>)`

---

## L√∂sung

```python
score = int(input("Enter score (0-100): "))

if score < 0 or score > 100:
    print("Invalid score.")
elif score >= 90:
    print("Grade: A (Pass)")
elif score >= 80:
    print("Grade: B (Pass)")
elif score >= 70:
    print("Grade: C (Pass)")
elif score >= 60:
    print("Grade: D (Fail)")
else:
    print("Grade: F (Fail)")
```

**Alternative (mit Variablen):**
```python
score = int(input("Enter score (0-100): "))

if score < 0 or score > 100:
    print("Invalid score.")
else:
    # Note bestimmen
    if score >= 90:
        grade = "A"
    elif score >= 80:
        grade = "B"
    elif score >= 70:
        grade = "C"
    elif score >= 60:
        grade = "D"
    else:
        grade = "F"
    
    # Pass/Fail bestimmen
    if grade in ["D", "F"]:
        status = "Fail"
    else:
        status = "Pass"
    
    print(f"Grade: {grade} ({status})")
```
