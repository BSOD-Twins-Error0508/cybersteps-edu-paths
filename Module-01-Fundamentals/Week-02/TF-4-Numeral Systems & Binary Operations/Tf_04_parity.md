## Notizen

- **Gelernt:** Parity Checking fÃ¼r Fehlererkennung, Even vs Odd Parity
- **Even Parity:** Gesamtanzahl 1en muss gerade sein (0, 2, 4, 6, 8)
- **Odd Parity:** Gesamtanzahl 1en muss ungerade sein (1, 3, 5, 7)
- **Limitation:** Parity erkennt Single-Bit-Fehler, aber nicht wenn 2+ Bits flippen
- **Anwendung:** Serielle Kommunikation (UART), RAM (ECC), DatenÃ¼bertragung
- **Modulo-Trick:** `count % 2 == 0` prÃ¼ft auf gerade Zahl
- **Alternative:** XOR aller Bits ergibt Parity Bit (XOR-Kette)
- **Tipp:** `str.count('1')` ist effizient fÃ¼r Parity-Berechnung in Python
