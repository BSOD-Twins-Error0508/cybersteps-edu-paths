The World Wide Web and Websites. Das World Wide Web und Webseiten.

On August 6, 1991, the first website was introduced to the world. Am 6. August 1991 wurde die erste Webseite der Welt vorgestellt.

Often, Internet and World Wide Web or the web are used interchangeably, but they are not the same thing. Oft werden Internet und World Wide Web oder das Web synonym verwendet, aber sie sind nicht dasselbe.

The Internet is the global network of computers, the infrastructure that allows information to travel. Das Internet ist das globale Netzwerk von Computern, die Infrastruktur, die es Informationen ermöglicht zu reisen.

The World Wide Web, on the other hand, is one of the many services that run on the Internet. Das World Wide Web hingegen ist einer der vielen Dienste, die auf dem Internet laufen.

It's a system of interlinked hypertext documents accessed via the Internet. Es ist ein System von miteinander verlinkten Hypertext-Dokumenten, auf die über das Internet zugegriffen wird.

Think of the Internet as the roads and the web as the system of addresses and destinations you can reach using those roads. Stell dir das Internet als die Straßen vor und das Web als das System von Adressen und Zielen, die du über diese Straßen erreichen kannst.

A web page is a single document, typically written in HTML, that can be displayed in a web browser. Eine Webseite ist ein einzelnes Dokument, typischerweise in HTML geschrieben, das in einem Webbrowser angezeigt werden kann.

A website is a collection of related web pages, images, videos, or other digital assets that are hosted on at least one web server, usually accessible via the Internet. Eine Website ist eine Sammlung von zusammenhängenden Webseiten, Bildern, Videos oder anderen digitalen Inhalten, die auf mindestens einem Webserver gehostet werden und normalerweise über das Internet zugänglich sind.

Websites can be broadly categorized. Webseiten können grob kategorisiert werden.

Static websites: These consist of fixed content. Statische Webseiten: Diese bestehen aus festem Inhalt.

Each page is coded in HTML and displays the same information to every visitor. Jede Seite ist in HTML codiert und zeigt jedem Besucher die gleichen Informationen.

They are simpler to create and host. Sie sind einfacher zu erstellen und zu hosten.

Dynamic websites: These can display different content and provide user interaction. Dynamische Webseiten: Diese können unterschiedliche Inhalte anzeigen und Benutzerinteraktion ermöglichen.

The content can be generated on the fly based on user actions, database information, or other parameters. Der Inhalt kann spontan generiert werden, basierend auf Benutzeraktionen, Datenbankinformationen oder anderen Parametern.

Think of social media sites, e-commerce stores, or news portals. Denk an Social-Media-Seiten, Online-Shops oder Nachrichtenportale.

For now, we'll focus on the building blocks that underpin both. Vorerst konzentrieren wir uns auf die Bausteine, die beiden zugrunde liegen.

How the Web Works: Browsers and Servers. Wie das Web funktioniert: Browser und Server.

When you visit a website, a few key components work together. Wenn du eine Webseite besuchst, arbeiten einige wichtige Komponenten zusammen.

Web Browser: This is the software on your computer or mobile device that you use to access the web, for example, Google Chrome, Apple Safari, Mozilla Firefox. Webbrowser: Das ist die Software auf deinem Computer oder Mobilgerät, die du zum Zugriff auf das Web verwendest, zum Beispiel Google Chrome, Apple Safari oder Mozilla Firefox.

Its main job is to fetch web pages from servers and display them to you. Seine Hauptaufgabe ist es, Webseiten von Servern abzurufen und sie dir anzuzeigen.

Web Server: This is a computer or a program running on a computer that stores website files, HTML documents, CSS stylesheets, images, and so on, and makes them available to browsers upon request. Webserver: Das ist ein Computer oder ein Programm, das auf einem Computer läuft und Webseitendateien speichert, HTML-Dokumente, CSS-Stylesheets, Bilder und so weiter, und sie auf Anfrage für Browser verfügbar macht.

This interaction is based on the client-server model. Diese Interaktion basiert auf dem Client-Server-Modell.

You, the user, type a web address, URL, like http://www.example.com, into your browser or click a link. Du, der Benutzer, gibst eine Webadresse, eine URL wie http://www.example.com, in deinen Browser ein oder klickst auf einen Link.

Your browser, the client, sends an HTTP, HyperText Transfer Protocol, request to the web server that hosts the website. Dein Browser, der Client, sendet eine HTTP-Anfrage, HyperText Transfer Protocol, an den Webserver, der die Webseite hostet.

This request asks for the specific web page. Diese Anfrage fordert die spezifische Webseite an.

The web server processes the request. Der Webserver verarbeitet die Anfrage.

If the requested page exists and is accessible, the server sends the page's files, primarily HTML, CSS, and JavaScript, back to the browser as an HTTP response. Wenn die angeforderte Seite existiert und zugänglich ist, sendet der Server die Dateien der Seite, hauptsächlich HTML, CSS und JavaScript, als HTTP-Antwort zurück an den Browser.

Your browser receives these files, interprets them, and renders, or draws, the web page on your screen. Dein Browser empfängt diese Dateien, interpretiert sie und rendert, also zeichnet, die Webseite auf deinem Bildschirm.

Introduction to HTML: Structuring the Web. Einführung in HTML: Strukturierung des Webs.

HTML stands for HyperText Markup Language. HTML steht für HyperText Markup Language.

It's the standard markup language used to create web pages. Es ist die Standard-Auszeichnungssprache, die zum Erstellen von Webseiten verwendet wird.

Markup language means it uses tags to define the structure and content of a document. Auszeichnungssprache bedeutet, dass sie Tags verwendet, um die Struktur und den Inhalt eines Dokuments zu definieren.

HTML tells the browser what each part of the page is: a heading, a paragraph, an image, a link, and so on. HTML teilt dem Browser mit, was jeder Teil der Seite ist: eine Überschrift, ein Absatz, ein Bild, ein Link und so weiter.

Core HTML Components. Grundlegende HTML-Komponenten.

Tags: These are keywords surrounded by angle brackets, like the p tag. Tags: Das sind Schlüsselwörter, die von spitzen Klammern umgeben sind, wie das p-Tag.

Tags usually come in pairs: an opening tag, for example h1, and a closing tag, for example, closing h1. Tags kommen normalerweise paarweise: ein öffnendes Tag, zum Beispiel h1, und ein schließendes Tag, zum Beispiel schließendes h1.

The closing tag has a forward slash before the tag name. Das schließende Tag hat einen Schrägstrich vor dem Tag-Namen.

Elements: An element consists of an opening tag, some content, and a closing tag. Elemente: Ein Element besteht aus einem öffnenden Tag, etwas Inhalt und einem schließenden Tag.

For example: opening p tag, This is a paragraph, closing p tag. Zum Beispiel: öffnendes p-Tag, Dies ist ein Absatz, schließendes p-Tag.

Here, the opening p tag is the opening tag, This is a paragraph is the content, and the closing p tag is the closing tag. Hier ist das öffnende p-Tag das öffnende Tag, Dies ist ein Absatz ist der Inhalt, und das schließende p-Tag ist das schließende Tag.

Some elements are empty or void, meaning they don't have content or a closing tag in the traditional sense, for example, the img tag for images, the br tag for line breaks. Einige Elemente sind leer oder void, was bedeutet, dass sie keinen Inhalt oder schließendes Tag im traditionellen Sinne haben, zum Beispiel das img-Tag für Bilder, das br-Tag für Zeilenumbrüche.

Attributes: These provide additional information about an HTML element and are always specified in the opening tag. Attribute: Diese liefern zusätzliche Informationen über ein HTML-Element und werden immer im öffnenden Tag angegeben.

Attributes come in name/value pairs like name equals value. Attribute kommen in Name-Wert-Paaren wie Name gleich Wert.

For example, in the anchor tag with href equals https://www.example.com, Visit Example, closing anchor tag, href is an attribute of the anchor or link tag, and its value is the URL. Zum Beispiel, im Anker-Tag mit href gleich https://www.example.com, Visit Example, schließendes Anker-Tag, ist href ein Attribut des Anker- oder Link-Tags, und sein Wert ist die URL.

Basic HTML Document Structure. Grundlegende HTML-Dokumentstruktur.

A typical HTML document has a fundamental structure. Ein typisches HTML-Dokument hat eine grundlegende Struktur.

DOCTYPE html: This declaration defines the document type to be HTML5. DOCTYPE html: Diese Deklaration definiert den Dokumenttyp als HTML5.

It's important for browsers to render the page correctly. Es ist wichtig für Browser, die Seite korrekt zu rendern.

The html tag: This is the root element of an HTML page. Das html-Tag: Das ist das Wurzelelement einer HTML-Seite.

All other elements go inside it. Alle anderen Elemente befinden sich darin.

The head tag: This element contains meta-information about the HTML document, which is not displayed on the page itself. Das head-Tag: Dieses Element enthält Meta-Informationen über das HTML-Dokument, die nicht auf der Seite selbst angezeigt werden.

This includes things like the page title, character set, links to CSS files, and so on. Dies umfasst Dinge wie den Seitentitel, den Zeichensatz, Links zu CSS-Dateien und so weiter.

The meta tag with charset equals UTF-8: Specifies the character encoding for the document. Das meta-Tag mit charset gleich UTF-8: Gibt die Zeichencodierung für das Dokument an.

UTF-8 is standard and supports a wide range of characters. UTF-8 ist Standard und unterstützt eine breite Palette von Zeichen.

The meta tag with name equals viewport and content equals width equals device-width, initial-scale equals 1.0: Configures the viewport for responsive web design, making the site adapt to different screen sizes like mobile phones. Das meta-Tag mit name gleich viewport und content gleich width gleich device-width, initial-scale gleich 1.0: Konfiguriert das Viewport für responsives Webdesign, damit sich die Seite an verschiedene Bildschirmgrößen wie Mobiltelefone anpasst.

The title tag: Sets the title of the web page, which appears in the browser tab or window title bar. Das title-Tag: Legt den Titel der Webseite fest, der im Browser-Tab oder in der Fenstertitelleiste erscheint.

The body tag: This element contains the visible page content, all the headings, paragraphs, images, links, and so on, that the user sees in the browser. Das body-Tag: Dieses Element enthält den sichtbaren Seiteninhalt, alle Überschriften, Absätze, Bilder, Links und so weiter, die der Benutzer im Browser sieht.

Common HTML Tags. Häufige HTML-Tags.

Here are a few common HTML tags you'll encounter. Hier sind einige häufige HTML-Tags, denen du begegnen wirst.

h1 to h6: Heading tags. h1 bis h6: Überschriften-Tags.

h1 is the most important heading, h6 is the least. h1 ist die wichtigste Überschrift, h6 die unwichtigste.

The p tag: Paragraph tag, for blocks of text. Das p-Tag: Absatz-Tag, für Textblöcke.

The a tag: Anchor tag, for creating hyperlinks. Das a-Tag: Anker-Tag, zum Erstellen von Hyperlinks.

The href attribute specifies the link's destination. Das href-Attribut gibt das Ziel des Links an.

Example: opening a tag with href equals https://www.google.com, Go to Google, closing a tag. Beispiel: öffnendes a-Tag mit href gleich https://www.google.com, Go to Google, schließendes a-Tag.

The img tag: Image tag, for embedding images. Das img-Tag: Bild-Tag, zum Einbetten von Bildern.

The src attribute specifies the path to the image, and the alt attribute provides alternative text if the image cannot be displayed. Das src-Attribut gibt den Pfad zum Bild an, und das alt-Attribut liefert alternativen Text, wenn das Bild nicht angezeigt werden kann.

Example: img tag with src equals image.jpg and alt equals A descriptive text for the image. Beispiel: img-Tag mit src gleich image.jpg und alt gleich Ein beschreibender Text für das Bild.

The div tag: Division tag. Das div-Tag: Division-Tag.

A generic container used to group other elements for styling or layout purposes. Ein generischer Container, der verwendet wird, um andere Elemente zu Styling- oder Layout-Zwecken zu gruppieren.

The span tag: Span tag. Das span-Tag: Span-Tag.

An inline container used to group text or other inline elements, often for styling a small piece of content within a larger block. Ein Inline-Container, der verwendet wird, um Text oder andere Inline-Elemente zu gruppieren, oft zum Stylen eines kleinen Inhaltsteils innerhalb eines größeren Blocks.

Introduction to CSS: Styling the Web. Einführung in CSS: Styling des Webs.

CSS stands for Cascading Style Sheets. CSS steht für Cascading Style Sheets.

While HTML provides the structure and content, CSS is responsible for the presentation and appearance of the web page. Während HTML die Struktur und den Inhalt bereitstellt, ist CSS für die Präsentation und das Erscheinungsbild der Webseite verantwortlich.

It controls aspects like colors, fonts, spacing, layout, and even animations. Es steuert Aspekte wie Farben, Schriftarten, Abstände, Layout und sogar Animationen.

The key principle here is separation of concerns: HTML is for what the content is, its meaning and structure, and CSS is for how the content looks, its visual presentation. Das Schlüsselprinzip hier ist die Trennung von Belangen: HTML ist dafür da, was der Inhalt ist, seine Bedeutung und Struktur, und CSS ist dafür da, wie der Inhalt aussieht, seine visuelle Präsentation.

This makes your code cleaner, easier to maintain, and more flexible. Dies macht deinen Code sauberer, einfacher zu warten und flexibler.

Basic CSS Syntax. Grundlegende CSS-Syntax.

A CSS rule consists of two main parts. Eine CSS-Regel besteht aus zwei Hauptteilen.

Selector: This defines which HTML element or elements the rule applies to. Selektor: Dies definiert, auf welches HTML-Element oder welche Elemente die Regel angewendet wird.

Selectors can target elements by their tag name, for example p, class name, for example .my-class, ID, for example #my-id, and more. Selektoren können Elemente nach ihrem Tag-Namen ansprechen, zum Beispiel p, Klassennamen, zum Beispiel .my-class, ID, zum Beispiel #my-id, und mehr.

Declaration Block: This is enclosed in curly braces and contains one or more declarations, separated by semicolons. Deklarationsblock: Dieser ist in geschweiften Klammern eingeschlossen und enthält eine oder mehrere Deklarationen, getrennt durch Semikolons.

Each declaration includes a CSS property and a value, separated by a colon. Jede Deklaration enthält eine CSS-Eigenschaft und einen Wert, getrennt durch einen Doppelpunkt.

Example: color colon blue semicolon, property: color, value: blue. Beispiel: color Doppelpunkt blue Semikolon, Eigenschaft: color, Wert: blue.

This rule selects all p, paragraph, elements on the page and styles them to have navy blue text and a font size of 16 pixels. Diese Regel wählt alle p-, Absatz-, Elemente auf der Seite aus und stylt sie so, dass sie marineblauenText und eine Schriftgröße von 16 Pixeln haben.

How to Include CSS. Wie man CSS einbindet.

There are three main ways to add CSS to an HTML document. Es gibt drei Hauptmethoden, um CSS zu einem HTML-Dokument hinzuzufügen.

External CSS, Recommended: You write your CSS rules in a separate file with a .css extension, for example, styles.css. Externes CSS, Empfohlen: Du schreibst deine CSS-Regeln in eine separate Datei mit der Erweiterung .css, zum Beispiel styles.css.

Then, you link this file to your HTML document using the link tag inside the head section. Dann verlinkst du diese Datei mit deinem HTML-Dokument mithilfe des link-Tags innerhalb des head-Bereichs.

This is the most common and preferred method because it keeps your styles separate from your HTML, making them easier to manage and reuse across multiple pages. Dies ist die gebräuchlichste und bevorzugte Methode, weil sie deine Stile von deinem HTML trennt und sie dadurch einfacher zu verwalten und über mehrere Seiten hinweg wiederzuverwenden macht.

Internal CSS: You can embed CSS rules directly within the HTML document by placing them inside style tags, also within the head section. Internes CSS: Du kannst CSS-Regeln direkt im HTML-Dokument einbetten, indem du sie in style-Tags platzierst, ebenfalls innerhalb des head-Bereichs.

This can be useful for single-page websites or for page-specific styles, but for larger sites, external stylesheets are better. Dies kann für einseitige Webseiten oder für seitenspezifische Stile nützlich sein, aber für größere Seiten sind externe Stylesheets besser.

Inline CSS, Generally Avoid: You can apply styles directly to an individual HTML element using the style attribute. Inline-CSS, Generell Vermeiden: Du kannst Stile direkt auf ein einzelnes HTML-Element anwenden, indem du das style-Attribut verwendest.

While this works, it's generally discouraged because it mixes presentation with structure, making the HTML harder to read and maintain. Obwohl dies funktioniert, wird es generell nicht empfohlen, weil es Präsentation mit Struktur vermischt und das HTML schwerer lesbar und wartbar macht.

It also has higher specificity, which can make it harder to override with external or internal styles. Es hat auch eine höhere Spezifität, was es schwieriger machen kann, es mit externen oder internen Stilen zu überschreiben.

HTML and CSS Working Together. HTML und CSS arbeiten zusammen.

HTML and CSS are designed to work in tandem. HTML und CSS sind so konzipiert, dass sie zusammenarbeiten.

HTML structures the content, and CSS tells the browser how to display that structured content. HTML strukturiert den Inhalt, und CSS sagt dem Browser, wie dieser strukturierte Inhalt angezeigt werden soll.

The browser reads the HTML to understand the elements on the page and then applies the CSS rules that match those elements. Der Browser liest das HTML, um die Elemente auf der Seite zu verstehen, und wendet dann die CSS-Regeln an, die zu diesen Elementen passen.

For example, if your HTML has: opening h1 tag, Welcome!, closing h1 tag, and your linked CSS file has: h1, curly brace, color colon teal semicolon text-align colon center semicolon, closing curly brace, the browser will display the Welcome! heading in a teal color and centered on the page. Zum Beispiel, wenn dein HTML hat: öffnendes h1-Tag, Welcome!, schließendes h1-Tag, und deine verlinkte CSS-Datei hat: h1, geschweifte Klammer, color Doppelpunkt teal Semikolon text-align Doppelpunkt center Semikolon, schließende geschweifte Klammer, wird der Browser die Welcome!-Überschrift in einer türkisfarbenen Farbe und zentriert auf der Seite anzeigen.
