What are Unix and Linux? Was sind Unix und Linux?

Unix is a family of multitasking, multi-user computer operating systems that originated in the late 1960s and early 1970s at Bell Labs. Unix ist eine Familie von Multitasking- und Mehrbenutzer-Computerbetriebssystemen, die in den späten 1960er und frühen 1970er Jahren bei Bell Labs entstand.

Its design principles, portability, a hierarchical file system, plain text for data storage, and a powerful command-line interface, or CLI, were revolutionary and remain influential. Seine Designprinzipien, Portabilität, ein hierarchisches Dateisystem, einfacher Text für Datenspeicherung und eine leistungsstarke Befehlszeilenschnittstelle, oder CLI, waren revolutionär und bleiben einflussreich.

Linux, created by Linus Torvalds in the early 1990s, is a Unix-like operating system kernel. Linux, erstellt von Linus Torvalds in den frühen 1990er Jahren, ist ein Unix-ähnlicher Betriebssystem-Kernel.

What we commonly refer to as Linux is actually the Linux kernel combined with a vast collection of software, system utilities, libraries, and applications, many of which come from the GNU Project. Was wir üblicherweise als Linux bezeichnen, ist tatsächlich der Linux-Kernel kombiniert mit einer riesigen Sammlung von Software, Systemprogrammen, Bibliotheken und Anwendungen, von denen viele aus dem GNU-Projekt stammen.

This combination is more accurately called GNU slash Linux. Diese Kombination wird genauer GNU Schrägstrich Linux genannt.

Linux is open source, meaning its source code is freely available to use, modify, and distribute. Linux ist Open Source, was bedeutet, dass sein Quellcode frei verfügbar ist zum Nutzen, Modifizieren und Verteilen.

This has led to a vast ecosystem of Linux-based operating systems. Dies hat zu einem riesigen Ökosystem von Linux-basierten Betriebssystemen geführt.

Key characteristics of Linux systems: Open Source. Free to use, distribute, and modify. Hauptmerkmale von Linux-Systemen: Open Source. Frei zu nutzen, zu verteilen und zu modifizieren.

Stability and Reliability. Known for being robust, often powering servers that run for years without rebooting. Stabilität und Zuverlässigkeit. Bekannt dafür, robust zu sein und oft Server anzutreiben, die jahrelang ohne Neustart laufen.

Security. Strong permission model and active community contribute to its security. Sicherheit. Ein starkes Berechtigungsmodell und eine aktive Community tragen zu seiner Sicherheit bei.

Flexibility. Can be adapted for a wide range of devices, from embedded systems and smartphones, Android is Linux-based, to desktops and the world's most powerful supercomputers. Flexibilität. Kann für eine breite Palette von Geräten angepasst werden, von eingebetteten Systemen und Smartphones, Android ist Linux-basiert, bis hin zu Desktops und den leistungsstärksten Supercomputern der Welt.

Command-Line Interface, or CLI. The shell provides powerful control over the system. Befehlszeilenschnittstelle, oder CLI. Die Shell bietet leistungsstarke Kontrolle über das System.

Hierarchical File System. A tree-like structure starting from the root, forward slash. Hierarchisches Dateisystem. Eine baumartige Struktur, die von der Wurzel, Schrägstrich, ausgeht.

The Unix slash Linux Philosophy Die Unix Schrägstrich Linux Philosophie

This philosophy emphasizes creating simple, short, clear, modular, and extensible software. Diese Philosophie betont die Erstellung einfacher, kurzer, klarer, modularer und erweiterbarer Software.

Write programs that do one thing and do it well. Each tool should have a focused purpose. Schreiben Sie Programme, die eine Sache tun und sie gut machen. Jedes Werkzeug sollte einen fokussierten Zweck haben.

Write programs to work together. The output of one program should easily become the input for another, often using pipes, the vertical bar symbol. Schreiben Sie Programme, die zusammenarbeiten. Die Ausgabe eines Programms sollte leicht zur Eingabe für ein anderes werden, oft unter Verwendung von Pipes, dem vertikalen Balkensymbol.

Write programs to handle text streams, because that is a universal interface. Text is a versatile format for program communication. Schreiben Sie Programme, um Textströme zu verarbeiten, weil das eine universelle Schnittstelle ist. Text ist ein vielseitiges Format für Programmkommunikation.

What is a Linux Distribution? Was ist eine Linux-Distribution?

Because the Linux kernel is open source, many different organizations and communities have packaged it with various tools, desktop environments, and additional software to create complete operating systems. Da der Linux-Kernel Open Source ist, haben viele verschiedene Organisationen und Gemeinschaften ihn mit verschiedenen Werkzeugen, Desktop-Umgebungen und zusätzlicher Software gepackt, um vollständige Betriebssysteme zu erstellen.

These are known as Linux distributions, or distros. Diese sind bekannt als Linux-Distributionen oder Distros.

Each distribution might cater to different needs. Some are designed for beginners, others for servers, specific hardware, or particular tasks like cybersecurity. Jede Distribution könnte auf unterschiedliche Bedürfnisse zugeschnitten sein. Einige sind für Anfänger konzipiert, andere für Server, spezifische Hardware oder bestimmte Aufgaben wie Cybersicherheit.

Some popular Linux distributions include: Ubuntu. Very popular, especially for desktops and cloud computing, known for its ease of use. Based on Debian. Einige beliebte Linux-Distributionen umfassen: Ubuntu. Sehr beliebt, besonders für Desktops und Cloud Computing, bekannt für seine Benutzerfreundlichkeit. Basierend auf Debian.

Debian. A very stable and influential distribution, the basis for many others, including Ubuntu and Kali Linux. Debian. Eine sehr stabile und einflussreiche Distribution, die Grundlage für viele andere, einschließlich Ubuntu und Kali Linux.

Fedora. A cutting-edge distribution sponsored by Red Hat, often features the latest software. Fedora. Eine hochmoderne Distribution, gesponsert von Red Hat, bietet oft die neueste Software.

Linux Mint. Based on Ubuntu, aims to provide a more traditional desktop experience out of the box. Linux Mint. Basierend auf Ubuntu, zielt darauf ab, eine traditionellere Desktop-Erfahrung direkt nach der Installation zu bieten.

CentOS slash Rocky Linux slash AlmaLinux. Popular for servers, focused on stability and long-term support, binary-compatible with Red Hat Enterprise Linux. CentOS Schrägstrich Rocky Linux Schrägstrich AlmaLinux. Beliebt für Server, fokussiert auf Stabilität und Langzeitunterstützung, binärkompatibel mit Red Hat Enterprise Linux.

Core Components in a Linux System Kernkomponenten in einem Linux-System

The Linux Kernel. This is the core of any Linux distribution. Der Linux-Kernel. Dies ist der Kern jeder Linux-Distribution.

It manages the system's hardware, CPU, memory, storage devices, runs processes, and handles system calls from applications. Er verwaltet die Hardware des Systems, CPU, Speicher, Speichergeräte, führt Prozesse aus und bearbeitet Systemaufrufe von Anwendungen.

The Shell. The shell is a command-line interpreter that allows you to interact with the kernel by typing commands. Die Shell. Die Shell ist ein Befehlszeileninterpreter, der es Ihnen ermöglicht, mit dem Kernel zu interagieren, indem Sie Befehle eingeben.

Bash, or Bourne Again SHell. A very common default shell in many Linux distributions, including Ubuntu for many years. Bash, oder Bourne Again SHell. Eine sehr verbreitete Standard-Shell in vielen Linux-Distributionen, einschließlich Ubuntu seit vielen Jahren.

It's powerful and widely used for scripting. Sie ist leistungsstark und wird weithin für Skripterstellung verwendet.

Zsh, or Z Shell. Another powerful shell, gaining popularity. Zsh, oder Z Shell. Eine weitere leistungsstarke Shell, die an Beliebtheit gewinnt.

It's the default in macOS and Kali Linux now and can be easily installed on other Linux distributions. Sie ist jetzt die Standardeinstellung in macOS und Kali Linux und kann leicht auf anderen Linux-Distributionen installiert werden.

Other shells like sh, Bourne Shell, ksh, KornShell, and fish also exist. Andere Shells wie sh, Bourne Shell, ksh, KornShell und fish existieren ebenfalls.

You'll interact with the shell through a terminal emulator application, often just called Terminal. Sie werden mit der Shell über eine Terminal-Emulator-Anwendung interagieren, oft einfach Terminal genannt.

Package Management: apt on Kali Linux Paketverwaltung: apt auf Kali Linux

Linux distributions use package managers to simplify installing, updating, and removing software. Linux-Distributionen verwenden Paketmanager, um die Installation, Aktualisierung und Entfernung von Software zu vereinfachen.

A package is an archive containing all the files needed for a piece of software, plus metadata like its version and dependencies, other packages it needs to function. Ein Paket ist ein Archiv, das alle für ein Stück Software benötigten Dateien enthält, plus Metadaten wie seine Version und Abhängigkeiten, andere Pakete, die es zum Funktionieren benötigt.

Package managers download packages from repositories, servers hosting packages. Paketmanager laden Pakete von Repositories herunter, Servern, die Pakete hosten.

Kali Linux, and other Debian-based systems, uses APT, or Advanced Package Tool. Kali Linux und andere Debian-basierte Systeme verwenden APT, oder Advanced Package Tool.

Key apt commands, usually run with sudo for administrative privileges. Wichtige apt-Befehle, normalerweise mit sudo für administrative Privilegien ausgeführt.

sudo apt update. Refreshes the local list of available packages from repositories. Do this before installing or upgrading. sudo apt update. Aktualisiert die lokale Liste verfügbarer Pakete von Repositories. Tun Sie dies vor der Installation oder Aktualisierung.

sudo apt upgrade. Upgrades all currently installed packages to their newest versions. sudo apt upgrade. Aktualisiert alle derzeit installierten Pakete auf ihre neuesten Versionen.

sudo apt install package name. Installs a new package. sudo apt install Paketname. Installiert ein neues Paket.

sudo apt remove package name. Removes a package. sudo apt remove Paketname. Entfernt ein Paket.

sudo apt autoremove. Removes packages that were automatically installed to satisfy dependencies for other packages and are now no longer needed. sudo apt autoremove. Entfernt Pakete, die automatisch installiert wurden, um Abhängigkeiten für andere Pakete zu erfüllen und jetzt nicht mehr benötigt werden.

apt search keyword. Searches for available packages related to a keyword. apt search Schlüsselwort. Sucht nach verfügbaren Paketen, die mit einem Schlüsselwort zusammenhängen.

apt show package name. Shows detailed information about a package. apt show Paketname. Zeigt detaillierte Informationen über ein Paket.

Berechtigungen kontrollieren den Zugriff auf Dateien und Verzeichnisse für den Besitzer, u, Gruppe, g, und andere, o, mit Lese-, r, Schreib-, w, und Ausführungsberechtigungen, x.
