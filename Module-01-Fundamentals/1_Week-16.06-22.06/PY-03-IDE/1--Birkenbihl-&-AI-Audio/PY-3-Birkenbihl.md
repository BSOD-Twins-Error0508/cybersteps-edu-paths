What is an IDE? Was ist eine IDE?

An Integrated Development Environment, or IDE, is a software application that provides comprehensive facilities to computer programmers for software development. Eine integrierte Entwicklungsumgebung, oder IDE, ist eine Softwareanwendung, die Computerprogrammierern umfassende Möglichkeiten zur Softwareentwicklung bietet.

An IDE normally consists of at least a source code editor, build automation tools, and a debugger. Eine IDE besteht normalerweise mindestens aus einem Quellcode-Editor, Build-Automatisierungswerkzeugen und einem Debugger.

Think of it as a specialized workbench for writing code, equipped with tools that make the process faster, easier, and less error-prone. Stellen Sie es sich als spezialisierte Werkbank zum Schreiben von Code vor, ausgestattet mit Werkzeugen, die den Prozess schneller, einfacher und weniger fehleranfällig machen.

Key components you'll often find in an IDE. Wichtige Komponenten, die Sie häufig in einer IDE finden werden.

Source Code Editor. This is more advanced than a basic text editor, like TextEdit or Notepad. Quellcode-Editor. Dieser ist fortgeschrittener als ein einfacher Texteditor wie TextEdit oder Notepad.

It usually includes Syntax Highlighting. Er enthält normalerweise Syntaxhervorhebung.

This displays code elements, such as keywords, variables, strings, and comments, in different colors and fonts to improve readability and spot errors quickly. Diese zeigt Codeelemente wie Schlüsselwörter, Variablen, Zeichenketten und Kommentare in verschiedenen Farben und Schriftarten an, um die Lesbarkeit zu verbessern und Fehler schnell zu erkennen.

Auto-completion, also called IntelliSense, suggests code completions as you type, reducing typos and speeding up coding. Auto-Vervollständigung, auch IntelliSense genannt, schlägt Code-Vervollständigungen während Sie tippen vor, reduziert Tippfehler und beschleunigt das Programmieren.

It can suggest variable names, function names, and module contents. Es kann Variablennamen, Funktionsnamen und Modulinhalte vorschlagen.

Code Formatting automatically formats your code to adhere to style guidelines, making it neat and consistent. Code-Formatierung formatiert Ihren Code automatisch entsprechend Stilrichtlinien, wodurch er ordentlich und konsistent wird.

Build Automation Tools help automate the process of compiling, if necessary, linking, and packaging code. Build-Automatisierungswerkzeuge helfen dabei, den Prozess des Kompilierens, falls erforderlich, Verlinkens und Verpackens von Code zu automatisieren.

For interpreted languages like Python, this often involves tools for managing project dependencies and running scripts. Bei interpretierten Sprachen wie Python umfasst dies oft Werkzeuge zur Verwaltung von Projektabhängigkeiten und zum Ausführen von Skripten.

Debugger. A crucial tool for finding and fixing errors, or bugs, in your code. Debugger. Ein entscheidendes Werkzeug zum Finden und Beheben von Fehlern oder Bugs in Ihrem Code.

It allows you to run your code step-by-step. Er ermöglicht es Ihnen, Ihren Code Schritt für Schritt auszuführen.

You can set breakpoints to pause execution at specific lines. Sie können Haltepunkte setzen, um die Ausführung an bestimmten Zeilen anzuhalten.

You can inspect the values of variables at different points during execution. Sie können die Werte von Variablen an verschiedenen Punkten während der Ausführung untersuchen.

You can analyze the call stack, which is the sequence of function calls. Sie können den Call Stack analysieren, also die Abfolge von Funktionsaufrufen.

Integrated Terminal. Many IDEs include a built-in terminal or command-line interface, so you don't have to switch constantly between your editor and a separate terminal window to run commands or scripts. Integriertes Terminal. Viele IDEs enthalten ein eingebautes Terminal oder eine Befehlszeilenschnittstelle, sodass Sie nicht ständig zwischen Ihrem Editor und einem separaten Terminalfenster wechseln müssen, um Befehle oder Skripte auszuführen.

Version Control Integration. Tools to interact with version control systems like Git directly within the IDE, such as staging changes, committing, pushing, and pulling. Versionskontrollintegration. Werkzeuge zur Interaktion mit Versionskontrollsystemen wie Git direkt innerhalb der IDE, wie das Bereitstellen von Änderungen, Committen, Pushen und Pullen.

Extensions or Plugins. Most modern IDEs support extensions or plugins that add more features, support for other languages, or integration with other tools. Erweiterungen oder Plugins. Die meisten modernen IDEs unterstützen Erweiterungen oder Plugins, die weitere Funktionen, Unterstützung für andere Sprachen oder Integration mit anderen Werkzeugen hinzufügen.

Why Use an IDE for Python? Warum eine IDE für Python verwenden?

While you can write Python in any text editor, using an IDE offers significant advantages, especially as you tackle larger projects. Obwohl Sie Python in jedem Texteditor schreiben können, bietet die Verwendung einer IDE erhebliche Vorteile, besonders wenn Sie größere Projekte angehen.

Increased Productivity. Features like auto-completion, syntax highlighting, and integrated debugging significantly speed up the development process. Erhöhte Produktivität. Funktionen wie Auto-Vervollständigung, Syntaxhervorhebung und integriertes Debugging beschleunigen den Entwicklungsprozess erheblich.

Improved Code Quality. Syntax highlighting helps catch errors early. Debuggers make finding and fixing bugs systematic. Code formatting ensures consistency. Verbesserte Codequalität. Syntaxhervorhebung hilft, Fehler früh zu erkennen. Debugger machen das Finden und Beheben von Bugs systematisch. Code-Formatierung gewährleistet Konsistenz.

Easier Project Management. IDEs often provide ways to organize your project files and manage dependencies. Einfacheres Projektmanagement. IDEs bieten oft Möglichkeiten, Ihre Projektdateien zu organisieren und Abhängigkeiten zu verwalten.

Seamless Workflow. Having the editor, terminal, and debugger in one place streamlines the write-run-debug cycle. Nahtloser Arbeitsablauf. Editor, Terminal und Debugger an einem Ort zu haben, optimiert den Schreib-Ausführ-Debug-Zyklus.

Introducing Visual Studio Code, or VS Code. Wir stellen Visual Studio Code vor, oder VS Code.

There are many IDEs available for Python, such as PyCharm, Spyder, and Atom. Es gibt viele IDEs für Python, wie PyCharm, Spyder und Atom.

We will be using Visual Studio Code, or VS Code. Wir werden Visual Studio Code, oder VS Code, verwenden.

VS Code is a free, open-source, and highly popular code editor developed by Microsoft. VS Code ist ein kostenloser, quelloffener und sehr beliebter Code-Editor, der von Microsoft entwickelt wurde.

While technically a code editor, its extensive features and vast marketplace of extensions effectively turn it into a powerful, lightweight IDE suitable for many programming languages, including Python. Obwohl es technisch gesehen ein Code-Editor ist, machen seine umfangreichen Funktionen und der große Marktplatz an Erweiterungen ihn effektiv zu einer leistungsstarken, leichtgewichtigen IDE, die für viele Programmiersprachen geeignet ist, einschließlich Python.

Key features relevant to us are: Excellent Python support via an official extension. Wichtige Funktionen für uns sind: Hervorragende Python-Unterstützung durch eine offizielle Erweiterung.

Integrated terminal. Integriertes Terminal.

Built-in Git support. Eingebaute Git-Unterstützung.

Powerful debugging capabilities. Leistungsstarke Debugging-Funktionen.

Highly customizable with themes and extensions. Hochgradig anpassbar mit Themes und Erweiterungen.

Runs on macOS, Windows, and Linux. Läuft auf macOS, Windows und Linux.

Setup: Installing VS Code and Python Extension. Einrichtung: Installation von VS Code und der Python-Erweiterung.

Before our live session, please complete the following setup. Vor unserer Live-Sitzung vervollständigen Sie bitte die folgende Einrichtung.

Install Visual Studio Code. Go to the official VS Code download page at code.visualstudio.com/download. Installieren Sie Visual Studio Code. Gehen Sie zur offiziellen VS Code Download-Seite unter code.visualstudio.com/download.

Download the Mac Universal Stable Build. Laden Sie den Mac Universal Stable Build herunter.

Once downloaded, open the zip file. This will extract the Visual Studio Code app. Nach dem Download öffnen Sie die Zip-Datei. Dies extrahiert die Visual Studio Code App.

Drag Visual Studio Code app to your Applications folder. Ziehen Sie die Visual Studio Code App in Ihren Anwendungsordner.

Launch VS Code from your Applications folder. You might see a security warning as it's downloaded from the internet. Click Open. Starten Sie VS Code aus Ihrem Anwendungsordner. Sie sehen möglicherweise eine Sicherheitswarnung, da es aus dem Internet heruntergeladen wurde. Klicken Sie auf Öffnen.

Install the Python Extension. Launch VS Code. Installieren Sie die Python-Erweiterung. Starten Sie VS Code.

Look for the Extensions icon in the Activity Bar on the left side. It looks like four squares, with one flying off. Click it. Suchen Sie nach dem Erweiterungs-Symbol in der Aktivitätsleiste auf der linken Seite. Es sieht aus wie vier Quadrate, von denen eines abfliegt. Klicken Sie darauf.

In the search bar at the top of the Extensions view, type Python. In der Suchleiste oben in der Erweiterungsansicht tippen Sie Python ein.

Find the extension named simply Python published by Microsoft. It should be the first result. Finden Sie die Erweiterung mit dem Namen Python, veröffentlicht von Microsoft. Es sollte das erste Ergebnis sein.

Click the Install button for this extension. It might take a moment to install. Klicken Sie auf die Schaltfläche Installieren für diese Erweiterung. Die Installation kann einen Moment dauern.

Optional but Recommended: Install code command in PATH. Optional aber empfohlen: Installieren Sie den code-Befehl im PATH.

Open VS Code. Open the Command Palette using the keyboard shortcut Shift plus Command plus P. Öffnen Sie VS Code. Öffnen Sie die Befehlspalette mit der Tastenkombination Umschalt plus Befehl plus P.

Type Shell Command in the Command Palette search bar. Tippen Sie Shell Command in die Suchleiste der Befehlspalette ein.

Select the command Shell Command: Install code command in PATH. Wählen Sie den Befehl Shell Command: Install code command in PATH aus.

You might be prompted for your administrator password. Enter it. Sie werden möglicherweise nach Ihrem Administratorpasswort gefragt. Geben Sie es ein.

After installation, you can open files or folders in VS Code directly from your regular terminal by typing code followed by the filename or foldername. Nach der Installation können Sie Dateien oder Ordner in VS Code direkt von Ihrem regulären Terminal aus öffnen, indem Sie code gefolgt vom Dateinamen oder Ordnernamen eingeben.

For example, code my_project. Zum Beispiel: code my_project.

Explore the Interface Briefly. Take a moment to familiarize yourself with the main parts of the VS Code window. Erkunden Sie kurz die Benutzeroberfläche. Nehmen Sie sich einen Moment Zeit, um sich mit den Hauptbereichen des VS Code-Fensters vertraut zu machen.

Activity Bar on the far left includes File Explorer, Search, Source Control, Run and Debug, and Extensions. Die Aktivitätsleiste ganz links enthält Datei-Explorer, Suche, Quellcodeverwaltung, Ausführen und Debuggen sowie Erweiterungen.

Side Bar shows content based on the Activity Bar selection, such as your files. Die Seitenleiste zeigt Inhalte basierend auf der Auswahl in der Aktivitätsleiste an, wie Ihre Dateien.

Editor Group is the main area where you'll write code. You can open multiple files in tabs. Die Editor-Gruppe ist der Hauptbereich, in dem Sie Code schreiben werden. Sie können mehrere Dateien in Registerkarten öffnen.

Status Bar at the bottom shows information like line number, Python interpreter, and errors or warnings. Die Statusleiste unten zeigt Informationen wie Zeilennummer, Python-Interpreter und Fehler oder Warnungen an.

Panel is the bottom area that can show the Terminal, Debug Console, Problems, and Output. Das Panel ist der untere Bereich, der das Terminal, die Debug-Konsole, Probleme und Ausgaben anzeigen kann.

You can toggle it with View, then Terminal, or the shortcut Control plus backtick. Sie können es mit Ansicht, dann Terminal oder der Tastenkombination Steuerung plus Backtick umschalten.
