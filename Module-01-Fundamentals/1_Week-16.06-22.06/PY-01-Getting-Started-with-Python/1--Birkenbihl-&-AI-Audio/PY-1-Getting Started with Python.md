What is Python?
Was ist Python?

Python is a popular, powerful, and versatile programming language.
Python ist eine beliebte, leistungsstarke und vielseitige Programmiersprache.

Think of a programming language as a way for humans to give instructions to computers.
Stell dir eine Programmiersprache als eine Möglichkeit vor, mit der Menschen Anweisungen an Computer geben.

Just like we use human languages like English or German to communicate with each other, we use programming languages to communicate with machines.
Genau wie wir menschliche Sprachen wie Englisch oder Deutsch verwenden, um miteinander zu kommunizieren, nutzen wir Programmiersprachen, um mit Maschinen zu kommunizieren.

Why is Python so popular, especially in fields like cybersecurity?
Warum ist Python so beliebt, besonders in Bereichen wie Cybersicherheit?

Readable and Relatively Simple: Python's syntax, the rules for writing it, is designed to be clear and readable, almost like plain English in some cases.
Lesbar und relativ einfach: Pythons Syntax, die Regeln zum Schreiben, ist so gestaltet, dass sie klar und lesbar ist, in manchen Fällen fast wie normales Englisch.

This makes it easier to learn, write, and understand compared to many other languages.
Das macht es im Vergleich zu vielen anderen Sprachen einfacher zu lernen, zu schreiben und zu verstehen.

Versatile, Swiss Army Knife: Python can be used for a vast range of tasks.
Vielseitig, wie ein Schweizer Taschenmesser: Python kann für eine riesige Bandbreite an Aufgaben verwendet werden.

Web Development: Building websites and web applications.
Webentwicklung: Das Erstellen von Websites und Web-Apps.

Data Science and Machine Learning: Analyzing data, building predictive models, and artificial intelligence.
Data Science und Machine Learning: Das Analysieren von Daten, das Erstellen prädiktiver Modelle und künstliche Intelligenz.

Automation: Writing scripts to automate repetitive tasks, very useful in cybersecurity.
Automatisierung: Das Schreiben von Skripten, um wiederholende Aufgaben zu automatisieren, sehr nützlich in der Cybersicherheit.

Cybersecurity: Developing security tools, analyzing malware, automating security operations, penetration testing, and more.
Cybersicherheit: Das Entwickeln von Sicherheitstools, das Analysieren von Malware, das Automatisieren von Sicherheitsoperationen, Penetrationstests und mehr.

Game Development, Scientific Computing, and much more.
Spielentwicklung, wissenschaftliches Rechnen und vieles mehr.

Large Community and Libraries: Python has a huge, active community of developers worldwide.
Große Community und Bibliotheken: Python hat eine riesige, aktive Community von Entwicklern weltweit.

This means plenty of support, tutorials, and pre-written code, libraries or modules, that you can use to perform complex tasks without writing everything from scratch.
Das bedeutet viel Unterstützung, Tutorials und vorgefertigten Code, Bibliotheken oder Module, die du nutzen kannst, um komplexe Aufgaben zu erledigen, ohne alles von Grund auf zu schreiben.

In cybersecurity, there are many specialized libraries for tasks like network analysis, cryptography, and interacting with security tools.
In der Cybersicherheit gibt es viele spezialisierte Bibliotheken für Aufgaben wie Netzwerkanalyse, Kryptographie und die Interaktion mit Sicherheitstools.

Interpreted Language: Python code is generally executed line by line by an interpreter.
Interpretierte Sprache: Python-Code wird in der Regel Zeile für Zeile von einem Interpreter ausgeführt.

This is different from compiled languages like C++ or Java where the entire code is first translated into machine code before running.
Das unterscheidet sich von kompilierten Sprachen wie C++ oder Java, bei denen der gesamte Code zuerst in Maschinencode übersetzt wird, bevor er ausgeführt wird.

This often makes the development process faster and more flexible.
Das macht den Entwicklungsprozess oft schneller und flexibler.

In this course, you'll learn how to use Python as a tool to solve problems, automate tasks, and understand cybersecurity concepts more deeply.
In diesem Kurs lernst du, Python als Tool zu nutzen, um Probleme zu lösen, Aufgaben zu automatisieren und Konzepte der Cybersicherheit tiefer zu verstehen.

Setup: Verifying Python on Your Mac.
Einrichtung: Überprüfen von Python auf deinem Mac.

To start writing and running Python code, you need the Python interpreter installed.
Um mit dem Schreiben und Ausführen von Python-Code zu beginnen, brauchst du den Python-Interpreter installiert.

Modern macOS versions usually come with Python 3 pre-installed, which is what we need.
Moderne macOS-Versionen kommen normalerweise mit Python 3 vorinstalliert, was genau das ist, was wir brauchen.

Let's verify your setup.
Lass uns deine Einrichtung überprüfen.

Check Your Python 3 Version:
Überprüfe deine Python-3-Version:

Open the Terminal application.
Öffne die Terminal-Anwendung.

You can find it in Applications, Utilities, Terminal, or by searching for Terminal using Spotlight, Command + Space.
Du findest sie in Anwendungen, Dienstprogramme, Terminal, oder indem du mit Spotlight suchst, Command + Space.

Type the following command exactly and press Enter: python3 --version.
Gib den folgenden Befehl genau ein und drücke Enter: python3 --version.

You should see output like Python 3.x.y, for example, Python 3.9.6 or Python 3.11.5.
Du solltest eine Ausgabe wie Python 3.x.y sehen, zum Beispiel Python 3.9.6 oder Python 3.11.5.

Any version of Python 3 like 3.8, 3.9, 3.10, 3.11, 3.12, etc. is perfectly fine for this course.
Jede Version von Python 3 wie 3.8, 3.9, 3.10, 3.11, 3.12 usw. ist für diesen Kurs perfekt in Ordnung.

Make a note of the version you have.
Notiere dir die Version, die du hast.

If the command gives an error like command not found, or if it unexpectedly shows a version starting with 2.x.y, then Python 3 might not be correctly installed or accessible.
Wenn der Befehl einen Fehler wie command not found gibt oder unerwartet eine Version mit 2.x.y anzeigt, könnte Python 3 nicht korrekt installiert oder zugänglich sein.

In that case, proceed to the Optional: Installing Python 3 steps below.
In diesem Fall fahre mit den optionalen Schritten zur Installation von Python 3 unten fort.

Optional Installing/Updating Python 3:
Optional: Installieren/Aktualisieren von Python 3:

If the check above failed, or if you specifically want the absolute latest version, though the pre-installed one is usually fine, you can install it manually.
Wenn die obige Überprüfung fehlgeschlagen ist oder du speziell die allerneueste Version möchtest, obwohl die vorinstallierte meistens in Ordnung ist, kannst du sie manuell installieren.

Download: Go to the official Python website's download page for macOS: https://www.python.org/downloads/macos/
Download: Gehe zur offiziellen Python-Website auf die Download-Seite für macOS: https://www.python.org/downloads/macos/

Look for the latest stable release, for example, Python 3.12.3.
Suche nach der neuesten stabilen Version, zum Beispiel Python 3.12.3.

Avoid pre-releases.
Vermeide Vorabversionen.

Click the macOS 64-bit universal2 installer link to download the .pkg file.
Klicke auf den Link für den macOS 64-bit universal2 Installer, um die .pkg-Datei herunterzuladen.

Install: Open the downloaded .pkg file and follow the installation wizard.
Installiere: Öffne die heruntergeladene .pkg-Datei und folge dem Installationsassistenten.

Use the default settings and enter your password when prompted.
Verwende die Standardeinstellungen und gib dein Passwort ein, wenn du dazu aufgefordert wirst.

Verify After Install: Close the Terminal completely, Command + Q, and re-open it.
Überprüfe nach der Installation: Schließe das Terminal komplett, Command + Q, und öffne es neu.

Run python3 --version again.
Führe python3 --version erneut aus.

It should now show the version you just installed.
Es sollte nun die Version anzeigen, die du gerade installiert hast.

If you still have issues, note the error message for the live session.
Wenn du immer noch Probleme hast, notiere die Fehlermeldung für die Live-Session.

Running Python: The Interactive Interpreter, REPL.
Python ausführen: Der interaktive Interpreter, REPL.

One of the easiest ways to run simple Python code is using its interactive interpreter, also known as the REPL, Read-Eval-Print Loop.
Eine der einfachsten Wege, einfachen Python-Code auszuführen, ist die Nutzung des interaktiven Interpreters, auch bekannt als REPL, Read-Eval-Print Loop.

Start the REPL:
Starte den REPL:

Open your Terminal.
Öffne dein Terminal.

Type python3 and press Enter.
Gib python3 ein und drücke Enter.

You should see output similar to this, the exact version number will match what you have installed: Python 3.12.3 (v3.12.3:f6650f9ad7, Apr 9 2024, 08:18:48) [Clang 13.0.0 (clang-1300.0.29.30)] on darwin Type help, copyright, credits or license for more information. >>>
Du solltest eine ähnliche Ausgabe sehen, die genaue Versionsnummer passt zu der, die du installiert hast: Python 3.12.3 (v3.12.3:f6650f9ad7, Apr 9 2024, 08:18:48) [Clang 13.0.0 (clang-1300.0.29.30)] on darwin Type help, copyright, credits or license for more information. >>>

The >>> is the Python prompt.
Das >>> ist die Python-Eingabeaufforderung.

It means the interpreter is waiting for you to type a Python command.
Es bedeutet, dass der Interpreter darauf wartet, dass du einen Python-Befehl eingibst.

Read-Eval-Print Loop: The REPL works like this:
Read-Eval-Print Loop: Der REPL funktioniert so:

Read: It reads the Python command you type.
Read: Er liest den Python-Befehl, den du eingibst.

Eval: It evaluates, executes, the command.
Eval: Er bewertet, führt aus, den Befehl.

Print: If the command produces an output, it prints the result to the screen.
Print: Wenn der Befehl eine Ausgabe erzeugt, druckt er das Ergebnis auf den Bildschirm.

Loop: It then shows the >>> prompt again, waiting for the next command.
Loop: Dann zeigt er die >>>-Eingabeaufforderung wieder an und wartet auf den nächsten Befehl.

Try Some Commands: Type the following lines one by one, pressing Enter after each:
Probiere einige Befehle: Gib die folgenden Zeilen eine nach der anderen ein und drücke nach jeder Enter.

Basic math: 2 + 2 (Expected output: 4)
Grundlegende Mathe: 2 + 2 (Erwartete Ausgabe: 4)

Printing text: The print() command, called a function, is used to display output.
Text ausgeben: Der print()-Befehl, eine Funktion genannt, wird verwendet, um Ausgaben anzuzeigen.

Text needs to be enclosed in quotes, " or '.
Text muss in Anführungszeichen eingeschlossen werden, " oder '.

print("Hello, Cybersecurity!") (Expected output: Hello, Cybersecurity!)
print("Hello, Cybersecurity!") (Erwartete Ausgabe: Hello, Cybersecurity!)

A simple calculation with print: print(10 * 5) (Expected output: 50)
Eine einfache Berechnung mit print: print(10 * 5) (Erwartete Ausgabe: 50)

Exiting the REPL:
Den REPL beenden:

To leave the Python interpreter and go back to the normal terminal prompt, type exit() and press Enter, or press Control + D.
Um den Python-Interpreter zu verlassen und zur normalen Terminal-Eingabeaufforderung zurückzukehren, gib exit() ein und drücke Enter, oder drücke Control + D.

You should see your regular terminal prompt again, for example, yourusername@yourmac ~ %.
Du solltest deine reguläre Terminal-Eingabeaufforderung wieder sehen, zum Beispiel yourusername@yourmac ~ %.
