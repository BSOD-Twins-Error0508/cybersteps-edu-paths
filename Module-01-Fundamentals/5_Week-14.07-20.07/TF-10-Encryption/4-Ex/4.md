# Caesar's Codebreaker

_Lend me your ears, and your Python skills!_

**Goal**

Implement Caesar cipher encryption and decryption functions in Python and use them to decrypt a given ciphertext by brute-forcing the key.

**Instructions**

**Part 1: Implement Caesar Cipher Functions**

Write a Python script that includes two functions:

1. `caesar_encrypt(text, shift)`:
    - Takes a string `text` (plaintext) and an integer `shift` as input.
    - Encrypts the `text` using a Caesar cipher with the given `shift`.
    - The function should only shift uppercase English letters (A-Z).
    - It should preserve spaces and any other non-alphabetic characters as they are.
    - The shift should wrap around the alphabet.
    - Returns the encrypted string (ciphertext).
2. `caesar_decrypt(text, shift)`:
    - Takes a string `text` (ciphertext) and an integer `shift` as input.
    - Decrypts the `text` using a Caesar cipher with the given `shift`.
    - It should only operate on uppercase English letters (A-Z), preserve other characters, and handle wrapping.
    - Returns the decrypted string (plaintext).

**Implementation Notes:** To implement the shift and wrap-around logic, you'll need to work with character codes (e.g., ASCII values). Consider how to convert letters to numbers (0-25), apply the shift, handle the wrap-around using the modulo operator (`%`), and then convert back to letters.

**Part 2: Decrypt the Message**

The following ciphertext was encrypted using a Caesar cipher (a single, uniform shift for all letters): `Wh wg pshhsf hc qfsohs hvob hc zsofb! Qfsohwbu wg hvs sggsbqs ct zwts.`

The key (the shift value) is an integer between 1 and 25. Use your `caesar_decrypt` function to find the original plaintext message.

**Submission**

Submit:

1. Your complete Python script (`.py` file) containing the `caesar_encrypt` and `caesar_decrypt` functions, and any code you used for Part 2.
2. The decrypted plaintext message from Part 2.
3. The shift key (an integer from 1 to 25) that successfully decrypted the message

# Caesars Codeknacker

_Leih mir deine Ohren und deine Python-Kenntnisse!_

**Ziel**

Implementiere Caesar-Chiffrierungs- und -Dechiffrierungsfunktionen in Python und verwende sie, um einen gegebenen Chiffretext durch Brute-Force-Angriff auf den Schlüssel zu entschlüsseln.

**Anleitung**

**Teil 1: Caesar-Chiffrierungsfunktionen implementieren**

Schreibe ein Python-Skript, das zwei Funktionen enthält:

1. `caesar_encrypt(text, shift)`:
- Nimmt eine Zeichenkette `text` (Klartext) und eine ganze Zahl `shift` als Eingabe.
- Verschlüsselt den `text` mit einer Caesar-Verschlüsselung mit der vorgegebenen `shift`.
- Die Funktion sollte nur englische Großbuchstaben (A-Z) verschieben.
    - Leerzeichen und alle anderen nicht-alphabetischen Zeichen sollten unverändert bleiben.
- Die Verschiebung sollte sich um das gesamte Alphabet erstrecken.
- Gibt die verschlüsselte Zeichenfolge (Chiffretext) zurück.
2. `caesar_decrypt(text, shift)`:
- Nimmt eine Zeichenfolge `text` (Chiffretext) und eine ganze Zahl `shift` als Eingabe entgegen.
    - Entschlüsselt den `text` mit einer Caesar-Verschlüsselung mit der angegebenen `shift`.
- Sie sollte nur auf englische Großbuchstaben (A-Z) angewendet werden, andere Zeichen beibehalten und den Rundlauf berücksichtigen.
- Gibt die entschlüsselte Zeichenfolge (Klartext) zurück.

**Hinweise zur Implementierung:** Um die Verschiebungs- und Umbruchlogik zu implementieren, müssen Sie mit Zeichencodes (z. B. ASCII-Werten) arbeiten. Überlegen Sie, wie Sie Buchstaben in Zahlen (0-25) umwandeln, die Verschiebung anwenden, den Umbruch mit dem Modulo-Operator (`%`) handhaben und dann wieder in Buchstaben umwandeln können.

**Teil 2: Entschlüsseln Sie die Nachricht**

Der folgende Chiffretext wurde mit einer Caesar-Chiffre (eine einzige, einheitliche Verschiebung für alle Buchstaben) verschlüsselt: `Wh wg pshhsf hc qfsohs hvob hc zsofb! Qfsohwbu wg hvs sggsbqs ct zwts.`

Der Schlüssel (der Verschiebungswert) ist eine ganze Zahl zwischen 1 und 25. Verwenden Sie Ihre Funktion `caesar_decrypt`, um die ursprüngliche Klartextnachricht zu finden.

**Einreichung**

Reichen Sie Folgendes ein:

1. Ihr vollständiges Python-Skript (`.py`-Datei) mit den Funktionen `caesar_encrypt` und `caesar_decrypt` sowie allen Codes, die Sie für Teil 2 verwendet haben.
2. Die entschlüsselte Klartextnachricht aus Teil 2.
3. Den Verschiebungsschlüssel (eine ganze Zahl zwischen 1 und 25), mit dem die Nachricht erfolgreich entschlüsselt wurde.
