# Web Surfer

_Let's talk web server talk._

**Goal**

Write a Python client that connects to a real web server using its IP address and the standard HTTP port (80). HTTP (HyperText Transfer Protocol) is the foundation of data communication for the World Wide Web. Your client will send a basic request to fetch a webpage and print the beginning of the server's response.

**Instructions**

1. Find the IP address for a simple website like `example.com` using command-line tools (`ping` or `nslookup`).
    
2. Write a Python script (`http_client.py`) that creates a socket and `connect()`s to the IP address you found on **port 80**.
    
3. Construct a minimal HTTP/1.1 GET request string, ensuring correct formatting with `\\r\\n` line endings and the final blank line. Remember the `Host` header.
    
    ```
    GET / HTTP/1.1\\r\\nHost: example.com\\r\\nConnection: close\\r\\n\\r\\n
    ```
    
    _(Replace `example.com` with the actual hostname)_
    
4. Encode the request to bytes and `sendall()` it.
    
5. `recv()` the beginning of the response (e.g., 4096 bytes).
    
6. Decode the response bytes (using `errors='ignore'` for safety) and print it.
    
7. Close the socket.
    
8. Observe the output – look for the HTTP status line, headers, and the start of the HTML.
    

**Submission**

Submit your Python script (`http_client.py`) and a screenshot of the first 10-15 lines of the output you received from `example.com` (or your chosen simple website).

# Web Surfer

_Sprechen wir über Webserver._

**Ziel**

Schreiben Sie einen Python-Client, der sich über die IP-Adresse und den Standard-HTTP-Port (80) mit einem echten Webserver verbindet. HTTP (HyperText Transfer Protocol) ist die Grundlage der Datenkommunikation im World Wide Web. Ihr Client sendet eine einfache Anfrage zum Abrufen einer Webseite und gibt den Anfang der Antwort des Servers aus.

**Anweisungen**

1. Finden Sie die IP-Adresse einer einfachen Website wie `example.com` mit Hilfe von Befehlszeilentools (`ping` oder `nslookup`).

2. Schreiben Sie ein Python-Skript (`http_client.py`), das einen Socket erstellt und sich über `connect()` mit der gefundenen IP-Adresse auf **Port 80** verbindet.
    
3. Erstellen Sie eine minimale HTTP/1.1-GET-Anfragezeichenfolge und achten Sie dabei auf die korrekte Formatierung mit Zeilenenden „\\r\\n“ und einer abschließenden Leerzeile. Denken Sie an den „Host“-Header.
    
    ```
    GET / HTTP/1.1\\r\\nHost: example.com\\r\\nConnection: close\\r\\n\\r\\n
    ```
    
    _(Ersetzen Sie `example.com` durch den tatsächlichen Hostnamen)_
    
4. Kodieren Sie die Anfrage in Bytes und senden Sie sie mit `sendall()`.
    
5. Rufen Sie mit `recv()` den Anfang der Antwort ab (z. B. 4096 Bytes).
    
6. Dekodieren Sie die Antwortbytes (verwenden Sie zur Sicherheit `errors='ignore'`) und geben Sie sie aus.
    
7. Schließen Sie den Socket.
    
8. Beobachten Sie die Ausgabe – suchen Sie nach der HTTP-Statuszeile, den Headern und dem Anfang des HTML-Codes.


**Einreichung**

Reichen Sie Ihr Python-Skript (`http_client.py`) und einen Screenshot der ersten 10–15 Zeilen der Ausgabe ein, die Sie von `example.com` (oder einer anderen einfachen Website Ihrer Wahl) erhalten haben.
