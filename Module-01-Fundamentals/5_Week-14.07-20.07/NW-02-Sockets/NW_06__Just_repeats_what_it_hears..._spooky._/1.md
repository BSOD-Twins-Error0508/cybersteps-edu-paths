_Just repeats what it hears... spooky._

**Goal**

Write a basic server and client from scratch to create an "echo" service. The client should send a message, the server should receive it and send the _exact same message_ back to the client, and the client should receive and print the echoed message.

**Instructions**

1. **Create the Server Script (`echo_server.py`):**
    - Import the `socket` module.
    - Create a socket.
    - `bind` the socket to an address (e.g., `0.0.0.0`) and a port (e.g., 9999).
    - Put the socket into `listen`ing mode.
    - Print a message indicating the server is listening.
    - `accept` an incoming connection. This will give you a connection object and the client's address. Print who connected.
    - Use `recv()` on the connection object to receive data from the client. Store the received bytes.
    - Print the received message (after decoding).
    - **Add Echo Logic:** Immediately `sendall()` the _same received bytes_ back to the client over the same connection object.
    - `close()` the client connection.
    - `close()` the server's listening socket.
    - Include basic `try...except` error handling around the socket operations.
2. **Create the Client Script (`echo_client.py`):**
    - Import the `socket` module.
    - Create a socket.
    - Define the server's address (`127.0.0.1`) and the port your server is listening on.
    - `connect()` to the server's address and port. Print a confirmation message upon connection.
    - Define a message string to send (e.g., "Hello Echo!").
    - Encode the message to bytes and `sendall()` it to the server. Print the message you sent.
    - **Add Echo Logic:** Wait to `recv()` the response (the echoed message) from the server. Store the received bytes.
    - Decode the received bytes back into a string and print it, perhaps prefixed with "Server echoed: ".
    - `close()` the client socket.
    - Include basic `try...except` error handling.
3. **Test your scripts:** Run the `echo_server.py` first, then run `echo_client.py`. The client should print the message it originally sent, but received back from the server.

**Submission**

Submit your `echo_server.py` and `echo_client.py` files.

_Wiederholt einfach, was es hört... gruselig._

**Ziel**

Schreiben Sie einen einfachen Server und Client von Grund auf neu, um einen „Echo”-Dienst zu erstellen. Der Client sollte eine Nachricht senden, der Server sollte diese empfangen und _genau dieselbe Nachricht_ an den Client zurücksenden, und der Client sollte die wiederholte Nachricht empfangen und ausgeben.

**Anleitung**

1. **Erstellen Sie das Server-Skript (`echo_server.py`):**
- Importieren Sie das Modul `socket`.
- Erstellen Sie einen Socket.
- `Binden` Sie den Socket an eine Adresse (z. B. `0.0.0.0`) und einen Port (z. B. 9999).
    - Versetzen Sie den Socket in den `listen`-Modus.
- Drucken Sie eine Meldung aus, die angibt, dass der Server empfangsbereit ist.
- `accept` eine eingehende Verbindung. Dadurch erhalten Sie ein Verbindungsobjekt und die Adresse des Clients. Drucken Sie aus, wer sich verbunden hat.
    - Verwenden Sie `recv()` für das Verbindungsobjekt, um Daten vom Client zu empfangen. Speichern Sie die empfangenen Bytes.
- Geben Sie die empfangene Nachricht (nach der Dekodierung) aus.
- **Fügen Sie eine Echo-Logik hinzu:** Senden Sie sofort mit `sendall()` dieselben empfangenen Bytes über dasselbe Verbindungsobjekt an den Client zurück.
- Schließen Sie die Client-Verbindung mit `close()`.
    - Schließen Sie den Listening-Socket des Servers mit `close()`.
- Fügen Sie eine grundlegende `try...except`-Fehlerbehandlung für die Socket-Operationen hinzu.
2. **Erstellen Sie das Client-Skript (`echo_client.py`):**
- Importieren Sie das Modul `socket`.
- Erstellen Sie einen Socket.
    - Definieren Sie die Adresse des Servers (`127.0.0.1`) und den Port, auf dem Ihr Server lauscht.
- Stellen Sie mit `connect()` eine Verbindung zur Adresse und zum Port des Servers her. Geben Sie nach der Verbindung eine Bestätigungsmeldung aus.
- Definieren Sie eine zu sendende Nachrichtenzeichenfolge (z. B. „Hello Echo!“).
    - Kodieren Sie die Nachricht in Bytes und senden Sie sie mit `sendall()` an den Server. Geben Sie die gesendete Nachricht aus.
- **Fügen Sie Echo-Logik hinzu:** Warten Sie auf die Antwort (die zurückgesendete Nachricht) vom Server mit `recv()`. Speichern Sie die empfangenen Bytes.
    - Dekodieren Sie die empfangenen Bytes zurück in eine Zeichenkette und geben Sie diese aus, eventuell mit dem Präfix „Server echoed: “.
- Schließen Sie den Client-Socket mit `close()`.
- Fügen Sie eine grundlegende `try...except`-Fehlerbehandlung hinzu.
3. **Testen Sie Ihre Skripte:** Führen Sie zuerst `echo_server.py` aus, dann `echo_client.py`. Der Client sollte die ursprünglich gesendete, aber vom Server zurückerhaltene Nachricht ausgeben.

**Einreichen**

Reichen Sie Ihre Dateien `echo_server.py` und `echo_client.py` ein.
