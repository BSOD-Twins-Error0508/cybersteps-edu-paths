# Änderungsprotokoll

Alle wichtigen Änderungen an diesem Projekt werden in dieser Datei dokumentiert.

Das Format basiert auf [Keep a Changelog](https://keepachangelog.com/de/1.0.0/),
und dieses Projekt folgt [Semantic Versioning](https://semver.org/lang/de/).

---

## [Unveröffentlicht]

### Geplant
- JSON und CSV Ausgabeformate
- Echtzeit-Log-Überwachung
- Web-basiertes Dashboard
- Konfigurations-Datei-Support
- Multi-Threading-Unterstützung

## [0.2.0] - 2026-01-21

### Hinzugefügt
- Umfassende Projekt-Dokumentation
- Unit-Testing-Framework mit pytest
- Entwicklungsumgebungs-Setup-Scripts
- Code-Qualitäts-Tools (black, pylint, mypy)
- Erweitertes Fehlerbehandlungs-System
- Strukturierte Projektorganisation

### Geändert
- Codebase in modulare Struktur reorganisiert
- Kommandozeilen-Interface verbessert
- Dokumentation mit Beispielen aktualisiert
- README mit Entwicklungs-Richtlinien erweitert

### Behoben
- Speicherverwaltung für große Log-Dateien
- Zeitzonen-Handhabung beim Timestamp-Parsing
- Edge-Cases bei IP-Adressen-Extraktion

## Entwicklungshinweise

### Versionsnummerierung
- **Major (X.0.0)**: Breaking Changes
- **Minor (0.X.0)**: Neue Features, rückwärts kompatibel
- **Patch (0.0.X)**: Bug-Fixes, kleinere Verbesserungen

### Kategorien
- **Hinzugefügt**: Neue Features
- **Geändert**: Änderungen an existierenden Features
- **Veraltet**: Features, die entfernt werden
- **Entfernt**: Entfernte Features
- **Behoben**: Bug-Fixes
- **Sicherheit**: Sicherheits-Verbesserungen
