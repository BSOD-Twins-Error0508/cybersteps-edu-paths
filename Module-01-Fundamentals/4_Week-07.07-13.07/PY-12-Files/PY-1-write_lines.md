## LÃ¶sung

```python
def write_lines(filename, lines):
    """Schreibt eine Liste von Strings in eine Datei, jeder String in eine neue Zeile."""
    with open(filename, 'w') as f:
        for line in lines:
            f.write(line + '\n')
```
