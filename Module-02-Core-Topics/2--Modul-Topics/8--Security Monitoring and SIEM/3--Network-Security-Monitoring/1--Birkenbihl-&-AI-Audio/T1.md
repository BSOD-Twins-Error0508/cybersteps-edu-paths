What is Network Security Monitoring? Was ist Network Security Monitoring?

At its core, Network Security Monitoring is the practice of collecting, analyzing, and responding to network data to detect and investigate security threats. Im Kern ist Network Security Monitoring die Praxis, Netzwerkdaten zu sammeln, zu analysieren und darauf zu reagieren, um Sicherheitsbedrohungen zu erkennen und zu untersuchen.

Think of it as the security camera system for a computer network. Denken Sie daran wie an ein Überwachungskamerasystem für ein Computernetzwerk.

It's not about preventing attacks outright, but about gaining the visibility needed to detect them when they happen and to understand what happened after the fact. Es geht nicht darum, Angriffe direkt zu verhindern, sondern darum, die nötige Sichtbarkeit zu erlangen, um sie zu erkennen, wenn sie passieren, und zu verstehen, was im Nachhinein geschehen ist.

The primary goal of Network Security Monitoring is to answer critical questions like, is there an unauthorized user on our network? Das Hauptziel von Network Security Monitoring ist es, kritische Fragen zu beantworten wie, gibt es einen unbefugten Benutzer in unserem Netzwerk?

Is a compromised machine communicating with a malicious command and control server? Kommuniziert eine kompromittierte Maschine mit einem bösartigen Command-and-Control-Server?

Is an attacker attempting to exfiltrate, or steal, data? Versucht ein Angreifer, Daten zu exfiltrieren oder zu stehlen?

What was the exact sequence of events during a security incident? Was war die genaue Abfolge der Ereignisse während eines Sicherheitsvorfalls?

The Assume Breach Mentality. Die Assume-Breach-Mentalität.

Modern cybersecurity operates on a principle of assume breach. Moderne Cybersicherheit operiert nach dem Prinzip der angenommenen Kompromittierung.

This means that we accept that preventative measures, like firewalls and antivirus, can and will eventually fail. Das bedeutet, dass wir akzeptieren, dass präventive Maßnahmen wie Firewalls und Antivirenprogramme irgendwann versagen können und werden.

No defense is perfect. Keine Verteidigung ist perfekt.

Therefore, it's not a question of if an attacker will get in, but when. Daher ist es keine Frage, ob ein Angreifer eindringt, sondern wann.

Given this reality, detection and response become just as important as prevention. Angesichts dieser Realität werden Erkennung und Reaktion genauso wichtig wie Prävention.

Network Security Monitoring is the key practice that enables this detection and response capability. Network Security Monitoring ist die Schlüsselpraxis, die diese Erkennungs- und Reaktionsfähigkeit ermöglicht.

It provides the evidence needed to find intruders and understand their actions. Es liefert die Beweise, die nötig sind, um Eindringlinge zu finden und ihre Aktionen zu verstehen.

The Network Security Monitoring Visibility Triad, Key Data Sources. Die Network Security Monitoring Visibility Triad, Schlüsseldatenquellen.

To get a clear picture of network activity, we rely on three main types of data, often called the Network Security Monitoring Visibility Triad. Um ein klares Bild der Netzwerkaktivität zu erhalten, verlassen wir uns auf drei Haupttypen von Daten, die oft als Network Security Monitoring Visibility Triad bezeichnet werden.

Each provides a different level of detail and serves a different purpose. Jede bietet eine unterschiedliche Detailtiefe und dient einem anderen Zweck.

Number one, Full Packet Capture. Nummer eins, Full Packet Capture.

This is the most granular form of network data. Dies ist die detaillierteste Form von Netzwerkdaten.

Full Packet Capture, often stored in dot pcap files, is a bit for bit recording of every single packet that crosses a certain point in the network. Full Packet Capture, oft in Punkt-pcap-Dateien gespeichert, ist eine Bit-für-Bit-Aufzeichnung jedes einzelnen Pakets, das einen bestimmten Punkt im Netzwerk passiert.

It is the ultimate ground truth of what happened. Es ist die ultimative Grundwahrheit dessen, was passiert ist.

Pros, it contains every detail of a network conversation, including the application layer data, or payload. Vorteile, es enthält jedes Detail einer Netzwerkkonversation, einschließlich der Anwendungsschichtdaten oder Payload.

It is invaluable for deep dive forensic analysis to reconstruct events with one hundred percent certainty. Es ist unschätzbar wertvoll für tiefgehende forensische Analysen, um Ereignisse mit hundertprozentiger Sicherheit zu rekonstruieren.

For example, you could carve an entire file or image out of a packet capture if it was transferred over an unencrypted protocol. Zum Beispiel könnten Sie eine ganze Datei oder ein Bild aus einer Paketaufzeichnung extrahieren, wenn es über ein unverschlüsseltes Protokoll übertragen wurde.

Cons, it generates enormous amounts of data, making long term storage very expensive and impractical. Nachteile, es generiert enorme Datenmengen, was die Langzeitspeicherung sehr teuer und unpraktisch macht.

Analyzing terabytes of packet capture data is slow and cumbersome. Die Analyse von Terabytes an Paketaufzeichnungsdaten ist langsam und umständlich.

Crucially, if traffic is encrypted, for example using Transport Layer Security, the payload will be unreadable, significantly reducing the value of the capture. Entscheidend ist, dass wenn der Datenverkehr verschlüsselt ist, zum Beispiel mit Transport Layer Security, die Payload unlesbar sein wird, was den Wert der Aufzeichnung erheblich reduziert.

Number two, Network Flow Data. Nummer zwei, Network Flow Data.

Flow data is metadata about network conversations. Flow-Daten sind Metadaten über Netzwerkkonversationen.

It doesn't record the content of the conversations, but rather summarizes them. Sie zeichnen nicht den Inhalt der Konversationen auf, sondern fassen sie zusammen.

Think of it as a phone bill, it shows who you called, when you called, and for how long, but not what you said. Denken Sie daran wie an eine Telefonrechnung, sie zeigt, wen Sie angerufen haben, wann Sie angerufen haben und wie lange, aber nicht, was Sie gesagt haben.

Common flow protocols include NetFlow, sFlow, and IPFIX. Häufige Flow-Protokolle sind NetFlow, sFlow und IPFIX.

A typical flow record includes fields like source and destination IP address, source and destination port, protocol such as TCP or UDP, start and end time, total bytes and packets transferred, and TCP flags, for example SYN, FIN, RST. Ein typischer Flow-Datensatz umfasst Felder wie Quell- und Ziel-IP-Adresse, Quell- und Zielport, Protokoll wie TCP oder UDP, Start- und Endzeit, insgesamt übertragene Bytes und Pakete sowie TCP-Flags, zum Beispiel SYN, FIN, RST.

Pros, extremely lightweight and scalable. Vorteile, extrem leichtgewichtig und skalierbar.

It provides a high level, thirty thousand foot overview of all network traffic, making it excellent for identifying anomalies, baselining normal activity, and detecting large scale events like distributed denial of service attacks or network scanning. Es bietet einen hochrangigen Überblick über den gesamten Netzwerkverkehr aus der Vogelperspektive, was es hervorragend macht zum Identifizieren von Anomalien, zur Baseline normaler Aktivität und zur Erkennung großer Ereignisse wie Distributed-Denial-of-Service-Angriffen oder Netzwerk-Scans.

Cons, it lacks the payload and granular details. Nachteile, es fehlt die Payload und die detaillierten Einzelheiten.

You can see that an internal computer sent five gigabytes of data to an unknown IP in Russia, but you can't know what that data was. Sie können sehen, dass ein interner Computer fünf Gigabyte Daten an eine unbekannte IP in Russland gesendet hat, aber Sie können nicht wissen, was diese Daten waren.

Number three, Transaction Logs. Nummer drei, Transaction Logs.

Transaction logs are structured, high level descriptions of specific network activities, generated by analyzing the raw packets. Transaction Logs sind strukturierte, hochrangige Beschreibungen spezifischer Netzwerkaktivitäten, die durch Analyse der Rohpakete generiert werden.

They provide the perfect balance between the overwhelming detail of packet capture and the high level summary of flow data. Sie bieten die perfekte Balance zwischen den überwältigenden Details der Paketaufzeichnung und der hochrangigen Zusammenfassung von Flow-Daten.

These logs tell you what happened at the application layer in a human readable format. Diese Logs sagen Ihnen, was auf der Anwendungsschicht in einem für Menschen lesbaren Format passiert ist.

The most powerful tool for generating these logs is Zeek. Das mächtigste Werkzeug zur Generierung dieser Logs ist Zeek.

Zeek observes network traffic and produces detailed, tab separated log files for various protocols. Zeek beobachtet Netzwerkverkehr und produziert detaillierte, durch Tabulatoren getrennte Log-Dateien für verschiedene Protokolle.

For example, you'll get a dns dot log, http dot log, ssl dot log, and files dot log. Zum Beispiel erhalten Sie ein dns-Punkt-log, http-Punkt-log, ssl-Punkt-log und files-Punkt-log.

Without looking at a single packet, you can immediately tell that the client one ninety two dot one sixty eight dot one dot one hundred made a DNS query for evil dash c two dash server dot com and got the IP one zero four dot twenty eight dot one dot one hundred back. Ohne ein einziges Paket anzusehen, können Sie sofort erkennen, dass der Client eins neunzig zwei Punkt eins sechzig acht Punkt eins Punkt einhundert eine DNS-Abfrage für evil-Strich-c-zwei-Strich-server-Punkt-com gemacht hat und die IP eins null vier Punkt achtundzwanzig Punkt eins Punkt einhundert zurückbekommen hat.

This is incredibly powerful. Das ist unglaublich mächtig.
