APIs (Application Programming Interfaces): Some applications or cloud services offer APIs that allow the SIEM to pull log data directly.

APIs (Application Programming Interfaces): Einige Anwendungen oder Cloud-Dienste bieten APIs an, die es dem SIEM ermöglichen, Log-Daten direkt abzurufen.

File Monitoring: Watching specific log files for new entries.

Dateiüberwachung: Das Beobachten spezifischer Log-Dateien auf neue Einträge.

Normalization: Different systems generate logs in different formats.

Normalisierung: Verschiedene Systeme erzeugen Logs in unterschiedlichen Formaten.

A SIEM takes these varied formats and converts them into a common, standardized format.

Ein SIEM nimmt diese variierenden Formate und wandelt sie in ein gemeinsames, standardisiertes Format um.

This makes it much easier to analyze and compare events from disparate sources.

Das macht es viel einfacher, Ereignisse aus unterschiedlichen Quellen zu analysieren und zu vergleichen.

For example, a login event from Windows and a login event from a Linux server would be normalized to represent the same type of event, even though their original log entries look very different.

Zum Beispiel würde ein Login-Ereignis aus Windows und ein Login-Ereignis von einem Linux-Server normalisiert, um dasselbe Ereignistyp darzustellen, auch wenn ihre originalen Log-Einträge sehr unterschiedlich aussehen.

Correlation: This is where the magic happens.

Korrelationsanalyse: Hier passiert die Magie.

The SIEM analyzes normalized events from different sources to identify relationships and patterns that might indicate a security incident.

Das SIEM analysiert normalisierte Ereignisse aus verschiedenen Quellen, um Beziehungen und Muster zu identifizieren, die auf einen Sicherheitsvorfall hindeuten könnten.

For example, a failed login attempt on a server followed by a successful login from a different geographic location within a short timeframe might trigger an alert, even if neither event alone seems suspicious.

Zum Beispiel könnte ein fehlgeschlagener Login-Versuch auf einem Server, gefolgt von einem erfolgreichen Login aus einem anderen geografischen Ort in kurzer Zeit, einen Alarm auslösen, auch wenn keines der Ereignisse allein verdächtig wirkt.

This is done using correlation rules.

Das geschieht mit Hilfe von Korrelationsregeln.

Alerting: When a correlation rule is triggered, or a specific critical event is detected, the SIEM generates an alert.

Alarmierung: Wenn eine Korrelationsregel ausgelöst wird oder ein spezifisches kritisches Ereignis erkannt wird, erzeugt das SIEM einen Alarm.

These alerts notify security analysts of potential threats, allowing them to investigate and respond.

Diese Alarme benachrichtigen Sicherheitsanalysten über potenzielle Bedrohungen und ermöglichen es ihnen, zu untersuchen und zu reagieren.

Alerts can be sent via email, SMS, or integrated into incident response platforms.

Alarme können per E-Mail, SMS oder in Incident-Response-Plattformen integriert versendet werden.

Reporting and Dashboards: SIEMs provide powerful reporting capabilities for compliance audits, for example, showing who accessed what data and security posture assessments.

Berichterstattung und Dashboards: SIEMs bieten leistungsstarke Berichtsfunktionen für Compliance-Audits, zum Beispiel um zu zeigen, wer auf welche Daten zugegriffen hat, und für Bewertungen der Sicherheitslage.

Dashboards offer a visual, real-time overview of security events, trends, and alerts, making it easier for analysts to monitor the environment.

Dashboards bieten eine visuelle, Echtzeit-Übersicht über Sicherheitsereignisse, Trends und Alarme, was es Analysten erleichtert, die Umgebung zu überwachen.

Benefits of using a SIEM. Centralized Visibility: A single pane of glass for all security-relevant data.

Vorteile der Nutzung eines SIEM. Zentrale Sichtbarkeit: Ein einziges Fenster für alle sicherheitsrelevanten Daten.

Enhanced Threat Detection: Ability to detect complex attacks that span multiple systems and would not be visible by looking at individual logs.

Verbesserte Bedrohungserkennung: Die Fähigkeit, komplexe Angriffe zu erkennen, die mehrere Systeme umspannen und nicht sichtbar wären, wenn man nur individuelle Logs betrachtet.

Compliance Management: Helps organizations meet regulatory requirements by providing audit trails and reports.

Compliance-Management: Hilft Organisationen, regulatorische Anforderungen zu erfüllen, indem es Prüfpfade und Berichte bereitstellt.

Faster Incident Response: By quickly identifying and alerting on threats, SIEMs reduce the time it takes to detect and respond to incidents.

Schnellere Incident-Response: Indem es Bedrohungen schnell identifiziert und alarmiert, reduzieren SIEMs die Zeit, die benötigt wird, um Vorfälle zu erkennen und darauf zu reagieren.

Forensics and Investigation: Provides a historical record of events for post-incident analysis.

Forensik und Untersuchung: Bietet einen historischen Datensatz von Ereignissen für die Analyse nach einem Vorfall.

Popular SIEM Solutions: Splunk. While many SIEM solutions exist, Splunk is one of the most widely recognized and powerful platforms for collecting, indexing, and analyzing machine-generated data, including security logs.

Beliebte SIEM-Lösungen: Splunk. Während viele SIEM-Lösungen existieren, ist Splunk eine der am weitesten anerkannten und leistungsstärksten Plattformen zum Sammeln, Indexieren und Analysieren maschinengenerierter Daten, einschließlich Sicherheits-Logs.

It is often referred to as a data-to-everything platform due to its versatility beyond just security, but its SIEM capabilities are particularly strong.

Es wird oft als Data-to-Everything-Plattform bezeichnet aufgrund seiner Vielseitigkeit jenseits der reinen Sicherheit, aber seine SIEM-Fähigkeiten sind besonders stark.

Why Splunk? Splunk is a favorite among cybersecurity professionals for several reasons.

Warum Splunk? Splunk ist ein Favorit unter Cybersicherheitsprofis aus mehreren Gründen.

Powerful Search Language (SPL): Splunk's Search Processing Language (SPL) is incredibly flexible and allows analysts to perform complex queries, transformations, and statistical analysis on vast datasets.

Leistungsstarke Suchsprache (SPL): Splunks Search Processing Language (SPL) ist unglaublich flexibel und ermöglicht Analysten, komplexe Abfragen, Transformationen und statistische Analysen auf riesigen Datensätzen durchzuführen.

This enables deep dives into logs to uncover subtle attack patterns.

Das ermöglicht tiefe Eintauchen in Logs, um subtile Angriffsmuster aufzudecken.

Real-time Monitoring: Splunk can ingest and analyze data in real-time, providing immediate visibility into security events as they occur.

Echtzeit-Überwachung: Splunk kann Daten in Echtzeit aufnehmen und analysieren und bietet sofortige Sichtbarkeit auf Sicherheitsereignisse, während sie eintreten.

Scalability: It is designed to handle petabytes of data, making it suitable for organizations of all sizes, from small businesses to large enterprises.

Skalierbarkeit: Es ist so konzipiert, dass es Petabytes an Daten handhaben kann, was es für Organisationen aller Größen geeignet macht, von kleinen Unternehmen bis zu großen Konzernen.

Extensibility: Splunk has a rich ecosystem of apps and add-ons that extend its functionality, allowing integration with various security tools and data sources.

Erweiterbarkeit: Splunk hat ein reiches Ökosystem aus Apps und Add-ons, die seine Funktionalität erweitern und die Integration mit verschiedenen Sicherheits-Tools und Datenquellen ermöglichen.

Dashboards and Visualizations: Its intuitive interface allows for the creation of highly customizable dashboards, making it easy to visualize security posture, identify trends, and present findings.

Dashboards und Visualisierungen: Seine intuitive Oberfläche ermöglicht die Erstellung hochgradig anpassbarer Dashboards, was es einfach macht, die Sicherheitslage zu visualisieren, Trends zu identifizieren und Ergebnisse zu präsentieren.

Incident Response and Forensics: Splunk's ability to quickly search and correlate historical data makes it an invaluable tool for incident investigation and forensic analysis.

Incident-Response und Forensik: Splunks Fähigkeit, historische Daten schnell zu durchsuchen und zu korrelieren, macht es zu einem unschätzbaren Tool für die Untersuchung von Vorfällen und forensische Analysen.

Key Terminology. Event: A single, discrete occurrence recorded in a log, for example, "User 'admin' logged in".

Schlüsselbegriffe. Ereignis: Ein einzelnes, diskretes Vorkommnis, das in einem Log aufgezeichnet wird, zum Beispiel "Benutzer 'admin' hat sich eingeloggt".

Alert: A notification generated by the SIEM when a suspicious or critical event or pattern of events is detected.

Alarm: Eine Benachrichtigung, die vom SIEM erzeugt wird, wenn ein verdächtiges oder kritisches Ereignis oder ein Muster von Ereignissen erkannt wird.

Correlation Rule: A predefined logic or set of conditions that the SIEM uses to identify specific patterns of events that indicate a potential security threat.

Korrelationsregel: Eine vordefinierte Logik oder ein Satz von Bedingungen, die das SIEM verwendet, um spezifische Muster von Ereignissen zu identifizieren, die auf eine potenzielle Sicherheitsbedrohung hindeuten.

Dashboard: A graphical user interface that displays key security metrics, event trends, and alerts in a visual and easily digestible format.

Dashboard: Eine grafische Benutzeroberfläche, die Schlüssel-Sicherheitsmetriken, Ereignistrends und Alarme in einem visuellen und leicht verständlichen Format anzeigt.

Setup. For this lesson, we will be using Splunk Enterprise running in a Docker container.

Setup. Für diese Lektion verwenden wir Splunk Enterprise, das in einem Docker-Container läuft.

This allows you to set up a local Splunk instance without complex installations.

Das ermöglicht es Ihnen, eine lokale Splunk-Instanz ohne komplizierte Installationen einzurichten.

First, ensure Docker Desktop is installed on your macOS machine.

Zuerst stellen Sie sicher, dass Docker Desktop auf Ihrer macOS-Maschine installiert ist.

You can download it from the official Docker website.

Sie können es von der offiziellen Docker-Website herunterladen.

If you have an Apple Silicon (M1/M2/M3) Mac, Docker Desktop will automatically use Rosetta to run x86-64 (Intel) based Docker images if needed.

Falls Sie einen Apple-Silicon-Mac (M1/M2/M3) haben, wird Docker Desktop automatisch Rosetta verwenden, um x86-64 (Intel)-basierte Docker-Images auszuführen, falls nötig.

However, it is good practice to explicitly specify the platform when pulling images to ensure compatibility.

Es ist jedoch gute Praxis, die Plattform explizit anzugeben, wenn Images gezogen werden, um Kompatibilität zu gewährleisten.

Get Splunk container docker pull splunk/splunk:latest

Holen Sie den Splunk-Container mit docker pull splunk/splunk:latest

Run Splunk with license acceptance: docker run --platform=linux/amd64 -p 8000:8000 -p 8088:8088 -p 9997:9997 -e SPLUNK_START_ARGS=--accept-license -e SPLUNK_PASSWORD='YourStrongPass123!' --name splunk splunk/splunk:latest

Starten Sie Splunk mit Lizenzakzeptanz: docker run --platform=linux/amd64 -p 8000:8000 -p 8088:8088 -p 9997:9997 -e SPLUNK_START_ARGS=--accept-license -e SPLUNK_PASSWORD='YourStrongPass123!' --name splunk splunk/splunk:latest.

This YourStrongPass123! will be the Splunk admin user password, you can change it but make sure to document your new admin password.

Dieses YourStrongPass123! wird das Passwort für den Splunk-Admin-Benutzer sein, Sie können es ändern, aber stellen Sie sicher, dass Sie Ihr neues Admin-Passwort dokumentieren.

Password should follow the complexity levels: Password length at least 8 characters.

Das Passwort sollte die Komplexitätsstufen erfüllen: Passwortlänge mindestens 8 Zeichen.

Password should contain at least one upper case character, at least one lower case character, at least one number, at least one special character.

Das Passwort sollte mindestens einen Großbuchstaben, einen Kleinbuchstaben, eine Zahl und ein Sonderzeichen enthalten.

Checking Docker logs for Splunk: docker logs -f splunk

Überprüfen der Docker-Logs für Splunk: docker logs -f splunk.

Accessing Splunk http://localhost:8000

Zugriff auf Splunk über http://localhost:8000.

Herzlichen Glückwunsch! Sie haben nun eine lokale Splunk Enterprise-Instanz laufen.
