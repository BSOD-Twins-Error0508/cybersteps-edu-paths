_Alle Arbeit und kein Spiel macht Jack zu einem langweiligen Jungen. Lasst uns mit ein paar Hashes spielen._

**Ziel**

Führen Sie einen einfachen Wörterbuchangriff mit John the Ripper durch, um eine Reihe von einfachen Passwort-Hashes zu knacken.

**Anleitungen**

1. Erstellen Sie eine Datei mit dem Namen `hashes.txt` auf Ihrem Rechner. Fügen Sie darin die folgenden SHA-1-Hashes ein, einen pro Zeile:
    
    ```
    7c4a8d09ca3762af61e59520943dc26494f8941b
    6057f3324a0673243104c295f7823593973a75fc
    5baa61e4c9b93f3f0682250b6cf8331b7ee68fd8
    
    ```
    
2. Verwenden Sie den Befehl `john`, um diese Hashes zu knacken. Sie müssen das richtige Hash-Format für SHA-1 angeben.
    

**Einreichung**

Reichen Sie einen einzelnen Screenshot ein, der den Befehl "john" und die Ausgabe der geknackten Passwörter zeigt.
