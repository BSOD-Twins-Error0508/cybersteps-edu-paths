What is Kali Linux?
Was ist Kali Linux?

Kali Linux is a specialized Linux distribution built specifically for penetration testing, ethical hacking, and digital forensics.
Kali Linux ist eine spezialisierte Linux-Distribution, die speziell für Penetrationstests, ethisches Hacking und digitale Forensik entwickelt wurde.

It's based on Debian, a robust and stable Linux operating system.
Sie basiert auf Debian, einem robusten und stabilen Linux-Betriebssystem.

What makes Kali Linux special is that it comes pre-loaded with hundreds of powerful tools used by cybersecurity professionals worldwide, many of which can be used to test for or simulate the threat vectors we just discussed.
Was Kali Linux besonders macht, ist, dass es vorinstalliert mit Hunderten mächtiger Tools kommt, die von Cybersicherheitsprofis weltweit genutzt werden, von denen viele dazu dienen, Bedrohungsvektoren zu testen oder zu simulieren, die wir gerade besprochen haben.

It is imperative that Kali Linux and its tools are only used on systems you have explicit permission to test.
Es ist zwingend erforderlich, dass Kali Linux und seine Tools nur auf Systemen verwendet werden, für die du eine explizite Testgenehmigung hast.

Imagine a mechanic's toolbox, but for a cybersecurity expert.
Stell dir die Werkzeugkiste eines Mechanikers vor, aber für einen Cybersicherheitsexperten.

Instead of wrenches and screwdrivers, Kali Linux provides tools for network scanning, vulnerability analysis, password cracking, web application testing, wireless attack analysis, and much more.
Statt Schraubenschlüsseln und Schraubenziehern bietet Kali Linux Tools für Netzwerkscans, Schwachstellenanalysen, Passwortknacken, Webapplikationstests, Analysen drahtloser Angriffe und vieles mehr.

Having all these tools in one place, configured to work together, saves security professionals a massive amount of time and effort.
Alle diese Tools an einem Ort zu haben, konfiguriert, um zusammenzuarbeiten, spart Sicherheitsprofis eine enorme Menge an Zeit und Aufwand.

Open Source.

Industriestandard.

Using Kali Linux allows ethical hackers to simulate real-world attacks to identify weaknesses in systems and networks, ultimately helping organizations improve their security.
Die Nutzung von Kali Linux ermöglicht ethischen Hackern, reale Angriffe zu simulieren, um Schwächen in Systemen und Netzwerken zu identifizieren und letztendlich Organisationen zu helfen, ihre Sicherheit zu verbessern.
