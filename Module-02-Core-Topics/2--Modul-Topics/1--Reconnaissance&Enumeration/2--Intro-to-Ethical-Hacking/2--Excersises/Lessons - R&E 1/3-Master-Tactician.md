- Kenne deinen Feind... indem du seine Taktik kennst!

**Ziel**

Sich mit dem Spektrum der MITRE ATT&CK® Enterprise Tactics vertraut machen, Unbekanntes identifizieren und eine unbekannte Taktik erforschen.

**Anleitung**

1. Navigieren Sie zur offiziellen MITRE ATT&CK®-Webseite für Enterprise Tactics: [https://attack.mitre.org/tactics/enterprise/](https://attack.mitre.org/tactics/enterprise/)
    
2. Überprüfen Sie die `Liste der angezeigten Enterprise Tactics` (z. B. Reconnaissance, Initial Access, Execution, Persistence usw.). `Dies sind die übergeordneten Ziele eines Gegners`.
    
3. Basierend auf Ihrem aktuellen Wissen (aus den Materialien vor dem Unterricht, aus der Lektion oder aus dem Allgemeinwissen):
    
    - Erstellen Sie **Liste A**: Schreiben Sie die Namen von 5 Taktiken auf, deren allgemeinen Zweck Sie bereits einigermaßen gut verstehen.
        1. **Aufklärung- Research**
        2. **Ressourcenentwicklung - Resource development**
        3. **Ausführung - Execution**
        4. **Befehl und Kontrolle - command and control**
        5. **Exfiltration**
    - Erstellen Sie **Liste B**: Schreiben Sie die Namen von 5 Taktiken auf, deren allgemeiner Zweck Ihnen unklar ist oder von denen Sie sich nicht erinnern, dass sie besprochen wurden und über die Sie mehr wissen möchten.
        1. **Entdeckung - Discovery**
        2. **Erstzugriff - initial access**
        3. **Ausführung - Execution**
        4. **Befehl und Kontrolle - command and control**
        5. **Exfiltration**
4. Wählen Sie **EINE** Taktik aus Ihrer "Liste B" aus, die Sie näher untersuchen möchten.
    
    `Meine Reihenfolge der Untersuchungen:`
    
    - **`Initial Access`**
    - **Execution**
    - **Discovery**
    - **Command and Control**
    - **Exfiltration**
5. Klicken Sie auf der MITRE ATT&CK®-Website auf die Taktik, die Sie aus Liste B ausgewählt haben, und lesen Sie die detaillierte Beschreibung.
    
    1. **`Initial Access` =** [TA0001](https://attack.mitre.org/tactics/TA0001),
6. Verfassen Sie in Ihren eigenen Worten eine kurze Zusammenfassung (ca. 3-5 Sätze), die erklärt:
    
    - Das Hauptziel des Gegners, wenn er diese Taktik einsetzt.
        - **Erstzugriff (Initial Access)**: Das Hauptziel eines Angreifers bei dieser Taktik ist es, sich **erstmalig Zugang zu einem Zielsystem** zu verschaffen, um anschließend weitere Aktivitäten auszuführen.
    - Beschreiben Sie kurz ein Beispiel für eine Technik, die unter diese Taktik fällt (Sie können die Techniken auf der Seite der Taktik finden).
        - Eine häufig genutzte Technik ist die Nutzung **externer Remote-Dienste**, bei der Angreifer legitime Zugänge wie VPNs, Remote-Desktop oder Cloud-Dienste kompromittieren, um ins interne Netzwerk einzudringen. Dadurch können sie das System unbemerkt betreten und den Grundstein für weitere Angriffe legen. Die Technik erfordert oft das Ausnutzen von Schwachstellen oder gestohlenen Zugangsdaten. Ziel ist immer der **Eintrittspunkt** für weitere böswillige Operationen.

**Einreichung**

1. Ihre "Liste A" (Namen von Taktiken, von denen Sie glauben, sie zu verstehen).
2. Ihre "Liste B" (Namen von Taktiken, die Ihnen weniger vertraut sind).
3. Der Name und die Taktik-ID (z. B. TAXXXX) der einzelnen Taktik, die Sie aus Liste B recherchiert haben.
4. Ihre kurze Zusammenfassung der recherchierten Taktik, einschließlich ihres Ziels und einer Beispieltechnik.
