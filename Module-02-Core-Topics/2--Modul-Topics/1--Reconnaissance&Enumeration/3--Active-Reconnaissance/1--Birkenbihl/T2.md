
Basic operators.
Grundlegende Operatoren.

Site colon domain dot com, restricts results to a specific website.
Site colon domain dot com, beschränkt die Ergebnisse auf eine spezifische Website.

Example, site colon example dot com login, searches for login only on example dot com.
Beispiel, site colon example dot com login, sucht nach login nur auf example dot com.

Filetype colon ext, searches for specific file types.
Filetype colon ext, sucht nach spezifischen Dateitypen.

Example, site colon example dot com filetype colon pdf confidential, searches for PDF files containing confidential on example dot com.
Beispiel, site colon example dot com filetype colon pdf confidential, sucht nach PDF-Dateien, die confidential enthalten, auf example dot com.

Inurl colon text, searches for URLs containing specific text.
Inurl colon text, sucht nach URLs, die spezifischen Text enthalten.

Example, inurl colon admin login, searches for URLs that have admin and login in them.
Beispiel, inurl colon admin login, sucht nach URLs, die admin und login enthalten.

Intitle colon text, searches for pages with specific text in their title.
Intitle colon text, sucht nach Seiten mit spezifischem Text in ihrem Titel.

Example, intitle colon index of slash site colon example dot com, might find directory listings.
Beispiel, intitle colon index of slash site colon example dot com, könnte Verzeichnislisten finden.

Related colon domain dot com, finds sites similar to the specified domain.
Related colon domain dot com, findet Seiten, die der angegebenen Domain ähnlich sind.

Link colon domain dot com, finds pages that link to the specified domain.
Link colon domain dot com, findet Seiten, die auf die angegebene Domain verlinken.

These operators can be combined to create highly specific queries.
Diese Operatoren können kombiniert werden, um hoch spezifische Anfragen zu erstellen.

Attackers use dorks to find exposed login pages, sensitive documents, error messages revealing software versions, directory listings, and more.
Angreifer nutzen Dorks, um exponierte Login-Seiten, sensible Dokumente, Fehlermeldungen, die Software-Versionen enthüllen, Verzeichnislisten und mehr zu finden.

Try it yourself.
Probiere es selbst aus.

Use your favorite search engine, Google, DuckDuckGo, etc., to find the robots dot txt file for a well-known, public website, for example, wikipedia dot org.
Nutze deine Lieblingssuchmaschine, Google, DuckDuckGo usw., um die robots dot txt Datei für eine bekannte, öffentliche Website zu finden, zum Beispiel wikipedia dot org.

Try a query like site colon wikipedia dot org filetype colon txt robots dot txt.
Probiere eine Anfrage wie site colon wikipedia dot org filetype colon txt robots dot txt.

WHOIS lookup.
WHOIS-Lookup.

When a domain name, like example dot com, is registered, the person or organization registering it must provide contact information.
Wenn ein Domain-Name, wie example dot com, registriert wird, muss die Person oder Organisation, die ihn registriert, Kontaktinformationen angeben.

This information is stored in a public database, and you can query it using a WHOIS lookup service.
Diese Informationen werden in einer öffentlichen Datenbank gespeichert, und du kannst sie mit einem WHOIS-Lookup-Service abfragen.

Information obtainable.
Erhältliche Informationen.

Registrant, name, organization, address, phone, email of the domain owner.
Registrant, Name, Organisation, Adresse, Telefon, E-Mail des Domain-Inhabers.

Administrative contact, contact details for administrative matters.
Administrativer Kontakt, Kontaktdetails für administrative Angelegenheiten.

Technical contact, contact details for technical matters.
Technischer Kontakt, Kontaktdetails für technische Angelegenheiten.

Domain registrar, the company that registered the domain, for example, GoDaddy, Namecheap.
Domain-Registrar, das Unternehmen, das die Domain registriert hat, zum Beispiel GoDaddy, Namecheap.

Name servers, NS records, the servers that handle DNS resolution for the domain.
Name-Server, NS-Records, die Server, die die DNS-Auflösung für die Domain handhaben.

Registration dates, when the domain was registered and when it expires.
Registrierungsdaten, wann die Domain registriert wurde und wann sie abläuft.

Limitations.
Einschränkungen.

Many domain owners now use WHOIS privacy or proxy registration services.
Viele Domain-Inhaber nutzen jetzt WHOIS-Privacy oder Proxy-Registrierungsdienste.

These services replace the actual owner's contact information with the details of a proxy service, making it harder to identify the true owner directly.
Diese Dienste ersetzen die tatsächlichen Kontaktinformationen des Inhabers durch Details eines Proxy-Dienstes, was es schwieriger macht, den wahren Inhaber direkt zu identifizieren.

However, sometimes even with privacy services, historical WHOIS records or patterns in registration can reveal clues.
Allerdings können manchmal, sogar mit Privacy-Diensten, historische WHOIS-Records oder Muster in der Registrierung Hinweise liefern.

Perform a WHOIS lookup on a non-sensitive, public domain, for example, wikipedia dot org.
Führe einen WHOIS-Lookup für eine unempfindliche, öffentliche Domain durch, zum Beispiel wikipedia dot org.

DNS enumeration.
DNS-Enumeration.

The domain name system, DNS, is like the phonebook of the internet.
Das Domain Name System, DNS, ist wie das Telefonbuch des Internets.

It translates human-readable domain names, for example, www dot example dot com, into machine-readable IP addresses, for example, 93.184.216.34.
Es übersetzt menschenlesbare Domain-Namen, zum Beispiel www dot example dot com, in maschinenlesbare IP-Adressen, zum Beispiel 93.184.216.34.

Passively querying DNS can reveal information about a target's infrastructure.
Passive Abfragen von DNS können Informationen über die Infrastruktur eines Ziels enthüllen.

Information obtainable passively.
Passiv erhältliche Informationen.

A records, map a hostname to an IP address.
A-Records, ordnen einen Hostnamen einer IP-Adresse zu.

AAAA records, map a hostname to an IPv6 address.
AAAA-Records, ordnen einen Hostnamen einer IPv6-Adresse zu.

MX records, mail exchange, specify the mail servers responsible for receiving emails for the domain.
MX-Records, Mail Exchange, geben die Mail-Server an, die für den Empfang von E-Mails für die Domain verantwortlich sind.

This tells you where their email is handled.
Das zeigt dir, wo ihre E-Mails gehandhabt werden.

NS records, name server, specify the authoritative name servers for the domain.
NS-Records, Name Server, geben die autoritativen Name-Server für die Domain an.

CNAME records, canonical name, alias one name to another.
CNAME-Records, Canonical Name, weisen einen Namen einem anderen zu.

TXT records, can contain arbitrary text, often used for verification purposes, for example, SPF records for email validation, domain ownership verification.
TXT-Records können beliebigen Text enthalten, oft für Verifizierungszwecke genutzt, zum Beispiel SPF-Records für E-Mail-Validierung, Domain-Besitzverifizierung.

SRV records, service, define the location, hostname and port number, of servers for specific services.
SRV-Records, Service, definieren den Ort, Hostnamen und Portnummer, von Servern für spezifische Dienste.

While tools like nslookup, available on Windows, macOS, Linux, and dig, common on macOS, Linux, can be used to query DNS servers directly, which can sometimes be logged by the target's DNS server, making it slightly less passive, many web-based DNS lookup tools act as intermediaries.
Während Tools wie nslookup, verfügbar auf Windows, macOS, Linux, und dig, üblich auf macOS, Linux, verwendet werden können, um DNS-Server direkt abzufragen, was manchmal vom DNS-Server des Ziels protokolliert werden kann und es etwas weniger passiv macht, fungieren viele web-basierte DNS-Lookup-Tools als Vermittler.

These tools query public DNS servers on your behalf, making your query indirect and thus more passive from the target's perspective.
Diese Tools fragen öffentliche DNS-Server in deinem Namen ab, was deine Anfrage indirekt macht und somit passiver aus der Perspektive des Ziels.
