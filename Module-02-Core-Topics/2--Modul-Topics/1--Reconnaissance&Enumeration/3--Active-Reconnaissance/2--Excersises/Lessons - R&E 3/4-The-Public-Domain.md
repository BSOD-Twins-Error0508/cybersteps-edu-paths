_Es muss mehr als [uber.com](http://uber.com/) geben!_

**Ziel**

Identifizieren Sie Webdomänen, die wahrscheinlich von Uber kontrolliert werden, abgesehen von "[uber.com](http://uber.com/)" und seinen direkten Subdomänen, indem Sie verschiedene passive Recherchemethoden anwenden.

**Anweisungen**

Ermitteln Sie andere Domains, die Uber gehören oder von Uber betrieben werden. Denken Sie dabei an ein breites Spektrum: Akquisitionen, Markenschutz (Rechtschreibfehler, TLDs), verschiedene Dienstleistungen (Eats, Freight), regionale Websites, WHOIS-Muster, Rechts-/Investorenseiten.

1. Nennen Sie so viele unterschiedliche Domains (keine Subdomains von "[Uber.com](http://uber.com/)"), von denen Sie glauben, dass sie von Uber kontrolliert werden.
2. Erläutern Sie kurz Ihre Argumentation oder die verwendete öffentliche Quelle/Methode für jede dieser Domains.
3. Warum ist die Identifizierung des gesamten Domainportfolios eines Unternehmens für die Sicherheitsbewertung so wichtig?

**Einreichung**

Reichen Sie ein Dokument ein, das Folgendes enthält:

- Ihre Liste der verschiedenen Domains, die wahrscheinlich von Uber kontrolliert werden.
- Für jede Domain Ihre Begründung/Quelle.
- Ihre Erklärung, warum ein vollständiges Domainportfolio wichtig ist.
