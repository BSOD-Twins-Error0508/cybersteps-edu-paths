| Verwendete Tools        | TryHackMe: Eine Online-Plattform für interaktives Lernen von Cybersicherheit durch gamifizierte Übungen und virtuelle Umgebungen.<br>Kali Linux: Eine spezialisierte Linux-Distribution für Penetrationstests und ethisches Hacking, vorinstalliert mit Sicherheits-Tools.<br>Metasploit Framework: Ein Open-Source-Tool zum Entwickeln, Testen und Ausführen von Exploits gegen Schwachstellen.<br>Google Dorking: Fortgeschrittene Suchtechniken in Suchmaschinen wie Google, um sensible Informationen zu finden.<br>WHOIS Lookup: Ein Tool zur Abfrage von Domain-Registrierungsdetails aus öffentlichen Datenbanken.<br>DNS Enumeration: Abfragen des Domain Name Systems, um Informationen wie IP-Adressen oder Mail-Server zu erhalten.<br>Shodan: Eine Suchmaschine für internetverbundene Geräte, die exponierte Dienste und Schwachstellen indexiert.<br>Wayback Machine: Ein digitales Archiv des Web, das historische Versionen von Websites zeigt.<br>Nmap: Ein Tool für Port-Scanning und Netzwerk-Erkundung (im Kontext aktiver Reconnaissance erwähnt). |
| Technische Fachbegriffe | Threat Vector (Bedrohungsvektor): Methode oder Weg, den ein Angreifer nutzt, um unbefugten Zugriff zu erlangen.<br>Reconnaissance (Reconnaissance): Phase des Sammelns von Informationen über ein Ziel vor einem Angriff.<br>Passive Reconnaissance: Sammeln von Informationen ohne direkte Interaktion mit dem Ziel.<br>Active Reconnaissance: Direkte Interaktion mit dem Zielsystem, z. B. durch Scanning.<br>Phishing: Betrügerische Versuche, sensible Daten durch Täuschung zu erlangen.<br>Malware: Schädliche Software, die Systeme infiziert oder schädigt.<br>Vulnerability (Schwachstelle): Schwäche in Software oder Systemen, die ausgenutzt werden kann.<br>Exploit: Code, der eine Schwachstelle ausnutzt.<br>Payload: Code, der nach einem erfolgreichen Exploit auf dem Ziel ausgeführt wird.<br>Social Engineering: Manipulation von Menschen, um Informationen oder Zugang zu erhalten.<br>Zero-Day Vulnerability: Unbekannte Schwachstelle ohne verfügbaren Patch.                                                                                  |
| Wichtige Vokabeln       | Cybersecurity (Cybersicherheit): Schutz von Systemen, Netzwerken und Daten vor digitalen Angriffen.<br>Ethical Hacking (Ethisches Hacking): Legales Testen von Systemen auf Schwachstellen mit Erlaubnis.<br>Penetration Testing (Penetrationstest): Simulierte Angriffe, um Sicherheitslücken zu finden.<br>Attack Surface (Angriffsfläche): Alle potenziellen Einstiegspunkte für Angriffe.<br>Patch: Update, das eine Schwachstelle behebt.<br>DNS (Domain Name System): System, das Domain-Namen in IP-Adressen umwandelt.<br>WHOIS: Protokoll zur Abfrage von Domain-Informationen.<br>Robots.txt: Datei auf Websites, die Crawler anweist, bestimmte Bereiche zu meiden.<br>Sitemap: XML-Datei mit Übersicht über Website-Inhalte.<br>Google Dorking: Spezielle Suchoperatoren in Google für gezielte Abfragen.                                                                                                                                                                                                                                                   |
