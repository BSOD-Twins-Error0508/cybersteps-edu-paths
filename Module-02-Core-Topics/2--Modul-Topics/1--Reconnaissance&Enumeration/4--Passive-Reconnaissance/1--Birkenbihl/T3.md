4. Operating System Fingerprinting.
5. Betriebssystem-Fingerprinting.

Identifying the operating system of the target can help tailor subsequent exploitation attempts, as vulnerabilities and attack techniques are often operating system-specific.
Das Identifizieren des Betriebssystems des Ziels kann helfen, nachfolgende Ausnutzungsversuche anzupassen, da Schwachstellen und Angriffstechniken oft betriebssystemspezifisch sind.

Nmap uses a variety of techniques for operating system fingerprinting, -O option, including analyzing responses to specific Transmission Control Protocol and User Datagram Protocol probes, Internet Control Message Protocol characteristics, and other network behaviors.
Nmap verwendet eine Vielzahl von Techniken für das Betriebssystem-Fingerprinting, Option -O, einschließlich der Analyse von Antworten auf spezifische Transmission-Control-Protocol- und User-Datagram-Protocol-Sondierungen, Internet-Control-Message-Protocol-Merkmale und andere Netzwerkverhalten.

It compares these responses to a database of known operating system fingerprints.
Es vergleicht diese Antworten mit einer Datenbank bekannter Betriebssystem-Fingerprints.

5. Traceroute.
6. Traceroute.

Traceroute tools map the network path between your machine and a target host by sending packets with increasing Time-To-Live values.
Traceroute-Tools kartieren den Netzwerkpfad zwischen deiner Maschine und einem Zielhost, indem sie Pakete mit steigenden Time-To-Live-Werten senden.

Each router along the path decrements the Time-To-Live.
Jeder Router entlang des Pfads verringert den Time-To-Live.

When Time-To-Live reaches zero, the router typically sends an Internet Control Message Protocol Time Exceeded message back.
Wenn Time-To-Live null erreicht, sendet der Router typischerweise eine Internet-Control-Message-Protocol-Time-Exceeded-Nachricht zurück.

This reveals the Internet Protocol addresses of intermediate routers.
Das enthüllt die Internet-Protocol-Adressen der intermediären Router.

This can help identify network boundaries, firewalls, and the target's Internet Service Provider.
Das kann helfen, Netzwerkgrenzen, Firewalls und den Internet-Service-Provider des Ziels zu identifizieren.

Common tools are traceroute on Linux/macOS and tracert on Windows.
Gängige Tools sind Traceroute auf Linux/macOS und Tracert auf Windows.

Ethical Considerations and Risks.
Ethische Überlegungen und Risiken.

Active reconnaissance is inherently more intrusive than passive methods.
Aktive Reconnaissance ist von Natur aus aufdringlicher als passive Methoden.

Detection, your Internet Protocol address will be visible to the target.
Entdeckung, deine Internet-Protocol-Adresse wird für das Ziel sichtbar sein.

Scanning activities can trigger Intrusion Detection Systems or Intrusion Prevention Systems.
Scan-Aktivitäten können Intrusion-Detection-Systeme oder Intrusion-Prevention-Systeme auslösen.

Logging, system administrators often log incoming connections and port scans.
Protokollierung, Systemadministratoren protokollieren oft eingehende Verbindungen und Port-Scans.

Legality, never perform active reconnaissance on any system or network without explicit, written permission from the owner.
Legalität, führe niemals aktive Reconnaissance auf einem System oder Netzwerk aus, ohne explizite, schriftliche Erlaubnis des Eigentümers.

Unauthorized scanning can be perceived as a hostile act and may have legal consequences.
Ungenehmigtes Scannen kann als feindlicher Akt wahrgenommen werden und rechtliche Konsequenzen haben.

Stability, very aggressive or poorly configured scans could potentially disrupt services on unstable or sensitive systems, although this is rare with modern tools like nmap using default settings.
Stabilität, sehr aggressive oder schlecht konfigurierte Scans könnten potenziell Dienste auf instabilen oder sensiblen Systemen stören, obwohl das bei modernen Tools wie Nmap mit Standardeinstellungen selten ist.
