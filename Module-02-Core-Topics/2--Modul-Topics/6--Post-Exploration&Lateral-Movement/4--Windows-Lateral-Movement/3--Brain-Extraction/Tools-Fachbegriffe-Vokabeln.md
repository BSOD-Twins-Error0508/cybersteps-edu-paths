Der Text behandelt Themen im Bereich der Cybersicherheit, speziell Lateral Movement in Windows-Umgebungen. Ich habe die Inhalte in die angegebenen Kategorien unterteilt: **Verwendete Tools**, **Technische Fachbegriffe** und **Wichtige Vokabeln**. Jede Kategorie listet die relevanten Begriffe auf, gefolgt von ihrer Bedeutung auf Deutsch. Die Ausgabe ist in einer Tabelle strukturiert für Klarheit.

| Kategorie / Begriff                      | Bedeutung                                                                                                                                                                                       |
| ---------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Verwendete Tools**                     |                                                                                                                                                                                                 |
| Mimikatz                                 | Ein Tool, das verwendet wird, um NTLM-Hashes aus dem Speicher eines kompromittierten Computers zu extrahieren, um Pass-the-Hash-Angriffe durchzuführen.                                         |
| PsExec                                   | Ein legitimes Kommandozeilen-Tool aus der Windows Sysinternals Suite, das Prozesse auf entfernten Systemen ausführt und von Angreifern für Lateral Movement missbraucht wird.                   |
| WMI (Windows Management Instrumentation) | Eine Infrastruktur für die Verwaltung von Daten und Operationen auf Windows-basierten Systemen, die von Angreifern genutzt wird, um Befehle auf entfernten Maschinen auszuführen, oft fileless. |
| WMIC                                     | Ein Kommandozeilen-Tool für WMI, das Prozesse auflistet oder ausführt, z. B. "wmic process call create".                                                                                        |
| RDP (Remote Desktop Protocol)            | Ein Protokoll für den Fernzugriff auf einen Computer, das Angreifern einen interaktiven Desktop ermöglicht, wenn gültige Anmeldeinformationen vorliegen.                                        |
| Net Share                                | Ein Befehl, um lokale Freigaben auf einer Maschine aufzulisten, einschließlich administrativer Shares wie C$ oder ADMIN$.                                                                       |
| Dir                                      | Ein Befehl, um den Inhalt eines Verzeichnisses anzuzeigen, z. B. für den Zugriff auf administrative Shares.                                                                                     |
| Echo                                     | Ein Befehl, um Text in eine Datei zu schreiben, z. B. zum Erstellen einer Batch-Datei für eine Payload.                                                                                         |
| **Technische Fachbegriffe**              |                                                                                                                                                                                                 |
| Lateral Movement                         | Die Menge von Techniken, die Angreifer verwenden, um sich von einer kompromittierten Maschine zu anderen im Netzwerk zu bewegen.                                                                |
| Beachhead                                | Die erste kompromittierte Maschine in einem Angriff, die als Ausgangspunkt dient, aber selten das Endziel ist.                                                                                  |
| Domain Controller                        | Ein Server in einem Active Directory-Netzwerk, der Authentifizierung und Autorisierung verwaltet; oft das ultimative Ziel eines Angreifers.                                                     |
| MITRE ATT&CK Framework (Tactic TA0008)   | Ein Rahmenwerk zur Beschreibung von Angriffstechniken; TA0008 bezieht sich auf die Phase des Lateral Movements.                                                                                 |
| Pass-the-Hash (PtH)                      | Eine Technik, bei der ein Angreifer NTLM-Hashes verwendet, um sich ohne das Klartext-Passwort zu authentifizieren.                                                                              |
| NTLM Hash                                | Ein einseitiger kryptografischer Hash eines Passworts, der in Windows für Authentifizierung verwendet wird.                                                                                     |
| Challenge-Response Protocol              | Ein Protokoll, bei dem ein Computer beweist, dass er ein Passwort kennt, ohne es über das Netzwerk zu senden.                                                                                   |
| Pass-the-Ticket (PtT)                    | Die Kerberos-Äquivalente zu Pass-the-Hash, bei der Kerberos-Tickets gestohlen und injiziert werden, um Zugriff zu erhalten.                                                                     |
| Kerberos                                 | Das Standard-Authentifizierungsprotokoll in modernen Active Directory-Umgebungen, sicherer als NTLM.                                                                                            |
| Key Distribution Center (KDC)            | Ein zentraler Server in Kerberos, der Tickets ausstellt und Authentifizierung handhabt.                                                                                                         |
| Ticket-Granting Ticket (TGT)             | Ein Kerberos-Ticket, das als Identitätsnachweis dient und für die gesamte Anmeldesitzung gespeichert wird.                                                                                      |
| Service Ticket                           | Ein Kerberos-Ticket für den Zugriff auf einen spezifischen Dienst, das vom KDC auf Basis des TGT ausgestellt wird.                                                                              |
| SMB (Server Message Block)               | Ein Protokoll für Dateifreigabe, das in Tools wie PsExec für den Zugriff auf administrative Shares verwendet wird.                                                                              |
| RPC (Remote Procedure Call)              | Ein Protokoll, das es einem Programm ermöglicht, Dienste auf einem anderen Computer anzufordern.                                                                                                |
| ADMIN$ Share                             | Eine versteckte administrative Freigabe auf Windows-Maschinen, die auf C:\Windows verweist.                                                                                                     |
| PSEXESVC.exe                             | Eine Service-Executable, die von PsExec hochgeladen und gestartet wird, um eine Shell auf dem Ziel zu erzeugen.                                                                                 |
| Fileless                                 | Eine Angriffsmethode, die keine Dateien auf der Festplatte des Ziels ablegt, z. B. bei WMI-Angriffen.                                                                                           |
| Event ID 4624                            | Ein Windows-Ereignisprotokoll-Eintrag für eine erfolgreiche RDP-Anmeldung.                                                                                                                      |
| **Wichtige Vokabeln**                    |                                                                                                                                                                                                 |
| Attacker                                 | Angreifer – Eine Person oder Entität, die ein System kompromittiert oder angreift.                                                                                                              |
| Credentials                              | Anmeldeinformationen – Daten wie Passwörter, Hashes oder Tickets, die für den Zugriff benötigt werden.                                                                                          |
| Hash                                     | Hash – Eine einseitige Funktion, die Daten in eine feste Länge umwandelt, z. B. für Passwörter.                                                                                                 |
| Ticket                                   | Ticket – Ein digitales Zertifikat in Kerberos für Authentifizierung.                                                                                                                            |
| Payload                                  | Nutzlast – Code oder Datei, die auf einem Ziel ausgeführt wird, um Schaden anzurichten oder Kontrolle zu erlangen.                                                                              |
| Foothold                                 | Standpunkt – Der anfängliche Zugriff auf ein System in einem Angriff.                                                                                                                           |
| Pivot                                    | Pivot – Der Wechsel von einer Maschine zu einer anderen im Netzwerk.                                                                                                                            |
| Exfiltrate                               | Exfiltrieren – Sensible Daten aus einem Netzwerk stehlen und herausbringen.                                                                                                                     |
| Ransomware                               | Ransomware – Schadsoftware, die Daten verschlüsselt und Lösegeld verlangt.                                                                                                                      |
| Persistent Control                       | Dauerhafte Kontrolle – Langfristiger Zugriff auf ein Netzwerk.                                                                                                                                  |
